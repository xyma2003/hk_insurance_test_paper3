# IIQE Paper 3 長期保險刷題

香港保險中介人資格考試 Paper III（長期保險）刷題工具，含768道模擬試題。

> **2026.07.18 卷三已 PASS** ✅

## 功能

### 📚 刷題模式
- 768題打亂順序
- 選對自動跳下一題
- 選錯提示正確答案
- 每隔5-10題自動從錯題本穿插一道錯題進來複習
- 可自由前後翻題，未答的題不會被標記為已答

### 📝 模擬考試
- 從768題中隨機抽取50題
- 每題即時反饋
- 考完出分（70分合格）

### 🤖 AI出題模考
- 50道AI新題（非題庫原題），基於官方手冊知識點出題
- 含組合題（i/ii/iii/iv）和否定題
- 每題答完顯示解析（附手冊章節依據）
- 考完可逐題回顧

### ❌ 錯題本
- 答錯自動收錄
- 答對累計3次自動移除
- 可單獨練習錯題
- 進度可視化（●●○ = 2/3）

## 使用方法

```bash
git clone https://github.com/xyma2003/hk-insurance-paper3.git
cd hk-insurance-paper3
open index.html
```

或直接下載 `index.html` 和 `questions.js`，雙擊打開即可。

> 數據存在瀏覽器 localStorage，關掉再打開進度還在。

## 文件說明

| 文件 | 說明 |
|------|------|
| `index.html` | Vue 3 單頁應用（刷題/模考/錯題本） |
| `questions.js` | 768道題結構化數據（題號/章節/問題/選項/答案） |
| `IIQE_Paper3_速通筆記.md` | 從官方研習手冊提煉的速通筆記 |

## 技術棧

- Vue 3（CDN，無需構建）
- 純前端，無後端
- localStorage 持久化

## 題目來源

從《長期保險 — 模擬試題2024年版》PDF 解析，共768題，覆蓋5個章節：

| 章節 | 主題 | 比重 |
|------|------|------|
| 1 | 人壽保險簡介 | 10% |
| 2 | 人壽保險及年金的種類 | 20% |
| 3 | 保險利益附約及其他產品 | 24% |
| 4 | 闡釋人壽保險單 | 24% |
| 5 | 人壽保險程序 | 22% |
