# 第1章 人壽保險簡介 (10%)

## 1.1 壽險定義與需求

### 定義
LOMA（美國壽險管理學會）定義：「保險公司承諾當被保險人死亡時支付保險金的保險。」

⚠️ **易考點**：這個定義「未能完全描述所有人壽保險合約」，因為現代壽險還提供儲蓄、投資、退休等功能，不只死亡賠償。考題會問這段引述是否完全正確——答案是「未能完全描述」。

### 壽險滿足的需求

**個人需求**：
- 受養人生活開支（最基本）
- 善終開支
- 教育經費
- 退休收入
- 償還按揭資金
- 應急資金
- 殘疾收入

**業務需求**：
- 關鍵人物（Key Person）—— 僱主為重要僱員購買，補償公司因該僱員死亡帶來的損失
- 企業擁有人
- 合夥人
- 僱員福利

⚠️ **易考點**：關鍵人物保險的投保人僱主、受保生命重要僱員、受益人公司。這是「對其他人的可保權益」的典型例子。

## 1.2 壽險原則

### 六大保險原則在人壽險中的適用情況

| 原則 | 人壽險適用？ | 原因解釋 |
|------|-------------|----------|
| **可保權益** | ✅ 適用 | 只在合約開始時需要，之後不再相干 |
| **最高誠信** | ✅ 適用 | 主動披露所有重要事實，不管保險人有否要求 |
| **近因** | ⚠️ 應用有限 | 壽險極少採用除外責任，死亡原因一般不重要；但自殺條款、意外附約時近因很重要 |
| **彌償** | ❌ 不適用 | 人命無法精確財務補償，壽險是**利益保單**(Benefit Policy)而非彌償保單 |
| **分擔** | ❌ 不適用 | 可擁有多份壽險保單，每份全額賠付，不會按比例分擔 |
| **代位權** | ❌ 不適用 | 壽險人賠付後無權向第三者追討（如車禍中死者的人壽保險人不能向肇事者追討） |

❌ **易錯點**：分擔原則不適用於壽險——擁有多份壽險保單完全正常，每份都全額賠付。但如果故意購買多份保單，理賠人員可能視為欺詐跡象。

❌ **易錯點**：代位權不適用於壽險——汽車保險人賠付後可向肇事者追討，但人壽保險人賠付死亡保險金後不能向第三者追討。

### 1.2.1 可保權益 (Insurable Interest)

#### 什麼是可保權益

可保權益是與保險標的（在壽險中指某人的生命）之間的某種關係，這種關係獲法律或衡平法認可，產生就該人生命購買保險的權利。

**核心邏輯**：如果你跟那個人沒有任何關係，你憑什麼就他的生命購買保險並因他死亡而獲益？這是源自常識的概念。

#### 《保險業條例》相關條文

| 條文 | 內容 | 關鍵記憶點 |
|------|------|-----------|
| **第64A條** | 父母/監護人對未成年人（未滿18歲）享有可保權益 | 這是**法定延伸**，是配偶以外**唯一**構成可保權益的血緣/家庭關係 |
| **第64B條** | 缺乏可保權益的保險合約屬**無效** | 法定要求源自此條 |
| **第64C條** | 具權益的人、合約為其使用/利益而訂立的人、合約為其訂立的人的名字**必須寫在合約中** | 實際上不限於直接受益人，包括保單所有人有意讓其收受益的人 |
| **第64D條** | 可追討數額不得超逾被保險人對受保生命的權益數額 | 只在壽險以**彌償方式**訂立時才有意義（如信用壽險） |

⚠️ **易考點**：第64A條是父母對**未成年子女**（<18歲）的可保權益。注意：是父母對子女，不是子女對父母。任何基於**其他血緣或家庭關係**所訂立的保險在技術上都是無效的——即在香港，兄弟姐妹之間、子女對父母、祖孫之間，**在沒有金錢利益關係的情況下**，不能構成可保權益。

❌ **易錯點**：不要把64A和64B搞混——64A是講父母對未成年人的可保權益（法定延伸），64B是講缺乏可保權益的合約無效（後果）。考題常反過來問。

#### 對誰有可保權益

| 關係 | 可保權益來源 | 金額限制 | 是否需要證明 |
|------|-------------|----------|-------------|
| **自己** | 司法假設 | 無限 | 不需證明 |
| **配偶** | 司法假設 | 無限 | 不需證明 |
| **未成年子女(<18歲)** | 第64A條法定延伸 | — | 不需證明 |
| **債務人** | 判例法（金錢利益） | 貸款金額+應計利息 | 須證明 |
| **生意合夥人** | 判例法（金錢利益） | 涉及個人服務時 | 須證明 |
| **合約關係對方** | 判例法（金錢利益） | 潛在財務損失 | 須證明 |
| **關鍵僱員** | 判例法（金錢利益） | 公司因僱員死亡的損失 | 須證明 |
| **其他血緣關係**（兄弟姐妹等） | ❌ 香港不承認 | — | — |

⚠️ **易考點**：對自己及配偶的可保權益是**司法假設**(judicial presumption)，無限金額，不必證明。對其他人的可保權益（除64A條延伸外）必須具備**可以用金錢來估值的權益**。

#### ⚠️ 何時需要可保權益？（關鍵問題）

**答案：只有在合約開始時才需要可保權益，之後可保權益不再相干。**

這是考試高頻考點，由此產生三個重要後果：

**1. 離婚後保單仍有效**
- 夫妻一方為配偶投保後，即使離婚，保單仍有效
- 投保方完全有權在適當時候收取保險利益
- **原因**：可保權益只在合約開始時需要，離婚後雖然配偶關係不再，但保單已經生效，不受影響

**2. 債務償還後保單仍有效**
- 你就債務人的生命投保後，即使債務已償還，保單仍有效
- 你仍可在適當時候「再次得到償付」
- **原因**：同上，保單開始時有可保權益即可

**3. 轉讓 (Assignment)**
- 保單所有人可把正當安排的壽險合約**轉讓給第三者**，即使該第三者對受保生命**缺乏可保權益**
- **但**：如果是為了繞過可保權益規定而轉讓 → **無效**
- 關鍵在於**買保單時的意圖**：
  - 買保單時有籠統的意圖把它轉讓 → 合法
  - 有意把它轉讓給一個對受保生命不具可保權益的**特定的人** → 無效（從起保日開始便屬無效）

❌ **易錯點**：轉讓後保單有效的前提是——轉讓的目的不是為了繞過可保權益的要求。如果目的就是繞過規定，合約從一開始就無效，因為事實上的被保險人（即預定承讓人）缺乏所需的可保權益。

#### 可保權益 vs 彌償的關係

第64D條規定可追討數額不得超逾權益數額——但這條文**只在壽險以彌償方式訂立時才有意義**。因為大多數壽險是利益保單，可保權益（對自己/配偶）是無限的，不可能超額賠償。信用壽險是例外，它是彌償保險，賠付限於貸款結欠。

### 1.2.2 披露責任 / 最高誠信 (Duty of Disclosure / Utmost Good Faith)

#### 什麼是重要事實 (Material Facts)

**定義**：「在訂定保費或決定是否承擔風險方面會影響一名審慎的保險人的判斷的任何情況」

⚠️ **易考點**：重要事實的判斷標準是**客觀的**——使用「審慎的保險人」標準，而非特定保險人的主觀標準。一項只被特定保險人視為重要的事實，不能真正構成重要事實。

#### 四個案例的核心結論

| 案例 | 事實摘要 | 結果 | 核心原則 |
|------|----------|------|----------|
| **個案一** | 投保人否認「過去三個月內疲勞過度超過一星期」，後因癌病死亡 | 保險人**敗訴** | 投保人所要披露的限於**重要事實**，並非任何被問及的事實；特定保險人認為重要不等於客觀重要 |
| **個案二** | 投保人未披露梗阻性睡眠窒息症病史，後患結腸癌索償 | 保險人**勝訴** | 沒有披露的疾病雖與索償疾病**無關**，但足以影響核保決定即可拒賠 |
| **個案三** | 投保人否認喝酒習慣（每天十罐啤酒），後死於舌癌 | 保險人**勝訴** | 同上——喝酒與舌癌是否直接相關**不重要**，未披露的重要事實足以拒賠 |
| **個案四** | 投保人已做體檢，但未披露既往心臟病史，後因主動脈瘤破裂死亡 | 保險人**勝訴** | 即使保險人已做體檢，投保人**仍有責任披露**病史；體檢不等於免責 |

⚠️ **易考點（最核心）**：索償人常辯稱「未披露的疾病與索償的疾病無關」——但**這種關係的存在不屬於援引最高誠信原則的先決條件**。即：不披露的事實不需要與索償原因有因果關係，只要它是重要事實（足以影響核保決定），保險人就有權拒賠。

❌ **易錯點**：個案二中投保人是巴士司機，20年來病症從未影響工作，還通過了公司每年體檢——但這些都不能免除披露責任。投訴委員會仍贊同保險人拒賠。

#### 不需要披露什麼

| 情況 | 原因 |
|------|------|
| 有理由不可能知道的情況 | 不能期望一般人自我診斷並作披露（如有些疾病對合資格醫生容易察覺，但不能期望一般人做到） |
| 免體檢投保中申請表/個人體檢表格沒有問及的事情 | 保險人沒有要求體檢，很難指稱投保人沒有披露表格沒問的事 |
| 要體檢投保中合資格醫療人員的疏忽遺漏或診斷錯誤 | 保險人不能因醫療人員的過失而為難投保人 |

#### 免體檢 vs 要體檢

| | 免體檢投保 (Non-medical) | 要體檢投保 (Medical) |
|---|---|---|
| **保險人要求** | 沒有要求體檢 | 要求體檢 |
| **披露範圍** | 保險人很難指稱申請表/體檢表沒問的事構成不披露 | 保險人不能因醫療人員疏忽遺漏/診斷錯誤而為難投保人 |
| **投保人責任** | 仍須如實回答申請表上的問題 | **仍有責任披露病史**（個案四），體檢不等於免責 |

⚠️ **易考點**：要體檢投保中，即使保險人已做體檢，投保人**仍有責任主動披露病史**。體檢的性質可能不足以揭露所有資料，除非該體檢足以揭露相關病史。

#### 身體檢查與私隱條例

- 保險人可要求合理身體檢查/測試補充口頭資料
- 進行任何測試前，**必須解釋要求該類資料的原因**（《個人資料（私隱）條例》）
- 受檢人有權了解測試結果

#### 不可異議條款 (Incontestability Provision)

- 香港大多數壽險保單包含此條款
- 在保單生效了一段時間（**可異議期**，通常2年）後，保險人不會對保單提出異議
- **例外**：除非有證據證實保單所有人的**欺騙行為**
- 詳情見第4章

### 1.2.3 其他保險原則（近因、彌償）

#### 近因 (Proximate Cause)

| 危險類型 | 定義 | 壽險中的應用 |
|----------|------|-------------|
| **受保危險** (Insured Perils) | 保單所承保的危險 | 壽險中除非自殺除外條款或意外死亡附約適用，否則死亡原因不重要 |
| **除外危險** (Excepted Perils) | 保單明確排除的危險 | 壽險保單甚少包含除外責任；**自殺**屬除外責任，但僅於免責期內 |
| **不保危險** (Uninsured Perils) | 既未列為受保也未列為除外的危險 | 壽險索償中不可能出現這類複雜情況 |

⚠️ **易考點**：近因原則在壽險中的應用比非壽險**有限**，因為壽險極少採用除外責任。近因主要在自殺條款和意外死亡附約中發揮作用。

#### 彌償 (Indemnity) 及其引伸

**為什麼彌償不適用於壽險？**
1. 保險收益絕不可能構成人命的精確財務補償 → 壽險是**利益保單**(Benefit Policies)
2. 不可能超額賠償，因為可保權益在大多數情況下是無限的

**彌償的兩個引伸（次原則）也不適用於壽險**：
- **分擔**：壽險不受此限，擁有多份保單正常，每份全額賠付
- **代位權**：壽險人賠付後不能向第三者追討

## 1.3 壽險保費的計算

### 定價標準

| 標準 | 含義 |
|------|------|
| **充足的** (Adequate) | 使保險人有金錢支付保險利益及履行合約中的其他責任 |
| **公平的** (Equitable/Fair) | 每位保單所有人繳納的保費須反映有關風險及約定利益 |

### 1.3.1a 三大定價因素

```
死亡率 + 利息 = 淨保費 (Net Premium / Pure Premium)
淨保費 + 附加保費 (Loading, 即開支) = 毛保費 (Gross Premium)
```

| 因素 | 作用 | 說明 |
|------|------|------|
| **死亡率** (Mortality) | 預計受保生命的死亡比率 | 核心！利用死亡表(mortality tables)預測；個別風險屬核保問題 |
| **利息** (Interest) | 保費收入投資回報 | 時間就是金錢；預期投資回報越高，保費可越低 |
| **開支** (Expenses) | 營運成本、佣金、稅款、異常死亡率可能性 | 附加保費(loading)應付所有預期及可能營運開支 |

⚠️ **易考點**：計算壽險保費的三大因素是**死亡率、利息、開支**——不是發病率。發病率是醫療/健康保險的考量因素。

### 1.3.1b 其他因素

- **分紅(PAR) vs 不分紅(NON-PAR)**：分紅保單保費較高，但可收紅利；不分紅保單保費較低
- 美式紅利(dividends)於週年宣布時支付；英式紅利(bonuses)通常是復歸的(reversionary)，支付保單利益時一起支付
- 保單一旦簽發，保費**不能更改**，保險人也不能取消或修訂合約

### 1.3.2 保費制度

#### 自然保費制度 vs 均衡保費制度

| 比較項目 | 自然保費制度 | 均衡保費制度 |
|----------|-------------|-------------|
| **保費** | 隨年齡增加而遞增 | 合約有效期內保持不變 |
| **早年** | 保費低（年輕死亡率低） | 保費相對風險「太高」 |
| **晚年** | 保費高（年長死亡率高） | 保費相對風險「太低」 |
| **現金價值** | 無 | 有（早年超額保費+利息形成儲備） |
| **逆選擇** | 易受影響（低風險者退出，高風險者續保） | 不受影響 |
| **現狀** | **已淘汰**（至少就真正長期保單而言） | **主流做法** |

⚠️ **易考點**：自然保費制度為何被淘汰？因為保費隨年齡變貴時，**低風險的人退出，高風險的人留下續保**（逆選擇/anti-selection），導致風險池不平衡，違反大數法則。

#### 均衡保費制度產生的四個特性

| 特性 | 說明 |
|------|------|
| **1. 現金價值/退保價值** | 早年「超額」保費+利息形成基金/儲備；退保價值 = 現金價值 - 退保費用 |
| **2. 保單抵押貸款** | 現金價值是貸款的可接受質押擔保；按現代保單條款，用現金價值抵押借錢是一種權利 |
| **3. 不能作廢** (Non-forfeiture) | 不繳保費時保單會失效(lapse)，但可利用現金價值維持有效 |
| **4. 清繳保險** (Paid-up insurance) | 不再繳保費但保單繼續有效，保額按淨現金價值下調（又稱減額清繳保單） |

❌ **易錯點**：定期壽險**沒有現金價值**——保費只按指定保險期內的死亡風險釐定。所以上述四個特性都不適用於定期壽險。

# 第2章 人壽保險及年金的種類 (20%)

## 2.1 傳統的人壽保險類別

### 三大基本形式概覽

| | 定期壽險 | 儲蓄壽險 | 終身壽險 |
|---|---|---|---|
| **保障期** | 指定期限 | 指定期限 | 終身（至99/100歲） |
| **現金價值** | **無** | 有 | 有 |
| **保費** | **最便宜** | 較貴（必賠） | 中等 |
| **賠付條件** | 期限內死亡才賠 | 死亡或期滿均賠 | 任何時候死亡都賠 |
| **又稱** | 短期人壽保險 | — | Whole of Life Insurance |

### 2.1.1 定期壽險 (Term Insurance)

#### 基本特點
- 僅為特定期間提供保障，又稱短期人壽保險(temporary life insurance)
- 保單利益限於：(a) 受保生命在特定期間內死亡；(b) 死亡發生時保單有效
- 大多數情況下到期時沒有索償 → **現有最便宜的保障形式**
- 理論上保險期可長可短（甚至幾小時），實際上短於一年的並不多見

---

#### 2.1.1a 定額 / 遞減 / 遞增定期壽險

| 類型 | 保額變化 | 保費變化 | 特點與應用 |
|------|----------|----------|-----------|
| **定額定期** (Level term) | 整個保險期內不變 | 年保費通常不變 | **最流行**；解決短暫保障需求（如不以分期償還的貸款） |
| **遞減定期** (Decreasing term) | 按年或其他周期遞減 | 年保費通常不變 | **最便宜**的壽險種類；適合逐漸減少的保障需求 |
| **遞增定期** (Increasing term) | 按年遞增（固定%或指數如CPI） | 按保額增加比例增加 | 抗通脹，維持購買力 |

⚠️ **易考點**：遞減定期壽險是**現有人壽保險種類中最便宜的**，因為保額不斷遞減且只有在保險期內死亡才賠付。

#### 遞減定期壽險的三個典型應用

| 類型 | 保障對象 | 受益人 | 保險性質 | 特點 |
|------|----------|--------|----------|------|
| **信用壽險** (Credit life insurance) | 借款人 | 貸款人（直接支付貸款結欠） | **彌償保險** | 通常以集體形式賣給貸款機構 |
| **家庭收入壽險** (Family income insurance) | 受保生命 | 受益人 | — | 每月支付指定金額，總支付金額按時遞減；可與其他保單結合 |
| **抵押贖回保險** (Mortgage redemption insurance) | 按揭人 | 承按人（有時被指名為受益人） | **利益保險** | 保額與抵押貸款遞減結欠相符；可用聯合壽險方式 |

#### ⚠️ 抵押贖回保險 vs 信用壽險（高頻考點）

| 比較 | 抵押贖回保險 | 信用壽險 |
|------|-------------|----------|
| **保障誰的權益** | 按揭人的權益 | 貸款人的權益 |
| **保險性質** | **利益保險**（即使死亡時債項已清還，保單仍須賠付） | 通常**彌償保險**（賠付限於貸款結欠） |
| **受益人** | 承按人有時要求被指名為受益人 | 貸款人 |

❌ **易錯點**：不要將抵押贖回保險與**按揭彌償保險**(Mortgage Indemnity Insurance)混淆。按揭彌償保險為承按人承保按揭貸款基於任何理由而無法收回的風險，是截然不同的產品。

#### 家庭收入壽險舉例

假設5年期家庭收入計劃，每月利益$1,000：
- 受保生命於**第4年年底**死亡 → 剩餘12個月 → 12次×$1,000 = $12,000
- 受保生命於**第50個月月底**死亡 → 剩餘10個月 → 10次×$1,000 = $10,000

⚠️ **易考點**：死亡越晚，剩餘支付期越短，總支付金額越小——所以是遞減定期。

---

#### 2.1.1b 可續保 / 可轉換定期壽險

| 類型 | 核心特權 | 毋須什麼 | 保費計算 | 逆選擇風險 | 常見限制 |
|------|----------|----------|----------|-----------|----------|
| **可續保** (Renewable) | 續保權利 | **毋須可保性(健康)證明** | 按**到達年齡**(attained age)計算新保費 | 有 | (1)續保保額≤原保額；(2)限制續保次數；(3)費率可能高於不可續保 |
| **可轉換** (Convertible) | 轉換為**永久計劃** | **毋須可保性(健康)證明** | 按到達年齡的標準費率計算 | 有 | (1)超過某年齡(如55或65歲)後不能轉換；(2)保單生效一段時間後不能轉換；(3)永久計劃保額以定期壽險保額為上限 |

⚠️ **易考點**：可續保和可轉換的共同特點是**毋須提供可保性（健康）證明**——這是最大的賣點，也是逆選擇風險的來源（健康狀況差的人更傾向行使這些權利）。

❌ **易錯點**：一年定期保單通常都是可續保的，一般稱為每年可續保定期壽險(YRT/ART)。可續保的保費會隨到達年齡增長而增加——不是保持不變。

---

### 2.1.2 儲蓄壽險 (Endowment Insurance)

- 承諾：受保生命活到指定期間結束時給付保額；若期間內死亡，即時給付保額
- **必定給付**（死亡或期滿二選一），所以保費不便宜
- **技術上** = 定期壽險 + 相同金額的純生存保險(pure endowment)
  - 純生存保險：只在受保生命保險期結束時仍生存才支付
- 分紅或不分紅均可
- **香港不流行**（保費率相對較高）

### 2.1.3 終身壽險 (Whole Life Insurance)

#### 基本特性
- 整段人生提供保障，又稱 whole of life insurance
- 不管受保生命何時去世，保單將於此時支付保額
- 當受保生命活到死亡表上**最長壽年歲**（一般是**99或100歲**）時，保險人也會支付保額，合約同時終止

⚠️ **易考點**：終身壽險在99/100歲時支付保額並終止合約——這**不是除外責任**，而是因為死亡表上的最長壽年歲到了盡頭，保險人按合約支付保額。所以終身壽險**必定給付**。

#### 三種繳費方式

| 繳費方式 | 保單名稱 | 特點 |
|----------|----------|------|
| **(i) 終身繳付** | 純粹壽險單(straight life) / 連續繳費終身壽險單(continuous premium whole life) | 終身繳保費 |
| **(ii) 限期內繳付** | — | 在受保生命生前的指定期間內繳清 |
| **(iii) 與年齡有關的限制** | — | 到達某年齡（如65歲）後毋須再繳；若在此前去世，毋須再繳 |

❌ **易錯點**：限期繳費和年齡限制繳費的區別——前者是指定**年期**（如20年），後者是指定**年齡**（如65歲）。如果在指定期間/年齡之前去世，均毋須再繳保費。

#### 其他特點
- 分紅或不分紅均可
- 可有變化，如等級保費保險單(graded-premium policy)：保費定期增加直至達到均衡保費水平

## 2.2 非傳統的人壽保險類別

### 2.2.1 萬用壽險 (Universal Life Insurance)

**四大特徵**：

| 特徵 | 說明 |
|------|------|
| **1. 靈活保費** | 首年保費須達最低限額；第二年後可停繳；現金價值不足應付60天費用及死亡收費時保單失效 |
| **2. 可調整死亡保險金** | 可要求增減；增加時可能須提供可保性證明 |
| **3. 分別列示定價** ("Unbundled" pricing) | 三個定價元素獨立披露：(i)保障純成本 (ii)利息 (iii)開支 |
| **4. 累積現金價值** | 繳付保費+利息收益，扣除營運開支收費及保險純成本後累積 |

**死亡保險金選擇**：
- 保額 + 現金價值（較低保費，因保險人須承擔更高給付風險）
- 只有保額（較高現金價值累積速率）

**年報**：保單所有人每年收到報告，包含死亡保險金選擇、有效保額、已付保費、已扣開支、保證及額外利息、已扣保險純成本、未償還貸款、現金價值提取及結存。

### 2.2.2 單位相連長期保險 (Unit-Linked Long Term Insurance)

- 又稱投資相連長期保單(Investment-Linked Long Term Policy)
- 保單價值直接與投資表現相連
- 全部或部分保費以當時基金價格購入基金單位
- **價值可升可跌**——消費者須意識到風險
- 可與多類型基金掛鈎：股票、固定利息投資、現金等
- 最常見的是終身壽險和儲蓄壽險

---

## 2.3 年金及退休金

### 定義

| 概念 | 定義 |
|------|------|
| **年金** (Annuity) | 保險人為得到預繳的一筆過付款或一連串付款（年金代價），承諾在某人生前或約定期限內，向指定個人週期性支付一連串款項（年金利益款項） |
| **退休金** (Pension) | 按月（或其他周期）向已退休的人提供收入利益，直至去世；可以包含年金 |

### 年金相關角色

| 角色 | 說明 |
|------|------|
| **合約持有人** / 年金購買人 | 預繳年金代價的人 |
| **年金標的人** (Annuitant) / 年金領取人 | 其生命決定年金支付期限的人 |
| **收款人** (Payee) | 收取年金利益的人 |

> 很多時候，收款人、年金標的人和合約持有人是同一人。

### 2.3.1 年金類型

#### 即期年金 vs 延期年金

| | 即期年金 (Immediate annuity) | 延期年金 (Deferred annuity) |
|---|---|---|
| **購買方式** | 通常整付（躉繳） | 整付或分期保費 |
| **開始支付** | 購買後第一個年金期完結時開始 | 從某特定時間或到達某特定年齡後開始 |
| **階段** | 無累積期 | **累積期**（繳費+投資增長）→ **年金期**（定期收取年金） |

#### 年金的變化形式

| 類型 | 支付條件 | 特點 |
|------|----------|------|
| **確定年金** (Annuity certain) | 僅於規定年期內定期支付，**不論年金標的人是否已死亡** | 與生死無關 |
| **終身年金** (Life annuity) | 於年金標的人**在世時**提供週期性付款 | 又稱 whole life annuity |
| **臨時終身年金** (Temporary life annuity) | 於指明年期內、年金標的人還活着時支付 | 有期限的終身年金 |
| **確定期間終身年金** (Life income annuity with period certain) / **保證年金** (Guaranteed annuity) | 至少於指定年期內支付利益（即使年金標的人已死亡）；若之後仍活着，終身支付 | 保證期+終身 |

⚠️ **易考點**：簡單年金計劃中，如果年金標的人在年金代價用盡前死亡，會「失去」已繳年金代價的餘額——這對公眾吸引力很小，所以年金在香港不常見。主要適合擁有相當資金而又沒有人需要贍養或沒有近親的老年人。

---

#### ⚠️ 年金核保與壽險核保的比較（高頻考點）

| 比較項目 | 人壽保險 | 年金 |
|----------|----------|------|
| **基礎** | **死亡率** | **生存率** |
| **年齡影響** | 年齡越大，保費越貴 | 年齡越大（開始領取時），每期年金越高 |
| **性別影響** | 同齡男性保費**高於**女性（男性死亡率較高） | 同齡男性年金**高於**女性（女性壽命較長） |
| **本質** | 越早死越可能賠付 | 越晚死越可能多收年金 |

**核心邏輯**：人壽保險以死亡率為基礎（越年輕越便宜），年金以生存率為基礎（開始領取時越年老，每期年金越高）——完全背道而馳！

❌ **易錯點**：年金的邏輯是反過來的——年金標的人活得越久，保險人付得越多，所以年齡越大開始領取，每期年金越高（因為預期支付期較短）。男性壽命較短，所以同齡男性每期年金高於女性。

---

### 2.3.1a 香港年金計劃 (HKMC Annuity Plan)

#### 概述
- 香港年金有限公司（香港按揭證券有限公司全資附屬公司）於**2018年7月5日**正式推出
- 屬於**公共即期終身年金計劃**
- 供**65歲或以上**的**香港永久性居民**認購
- 收取整付保費後，提供穩定的**保證每月年金款項**（固定金額）

#### 產品特點

| 項目 | 數字/內容 |
|------|-----------|
| **認購年齡** | ≥65歲 |
| **身份** | 香港永久性居民 |
| **保費範圍** | $50,000 - $3,000,000（整付） |
| **保證期** | 從保費起繳日開始，直至累計保證每月年金總額達到已繳保費的**105%**為止 |
| **保證期內身故** | 指定受益人可收每月身故賠償金直至累計款項達105%；或選擇一次性身故賠償金（保證現金價值 vs 已繳保費100%扣除已付年金，取較高者） |
| **退保** | 可在保證期內申請退保換取退保價值；**提早退保或會導致財務損失** |
| **特別款項提取** | 保證期內可申請，用於香港醫療/牙科開支；上限$300,000；只可一次；提取後每月年金額相應調低 |

⚠️ **易考點**：香港年金計劃的保證期是已繳保費的**105%**——不是100%，不是110%。這個數字常考。

⚠️ **易考點**：特別款項提取上限是$300,000，**只可透過單次申請使用一次**，不限於特定危疾（只要是醫生/牙醫認為有需要的醫療/牙科治療或檢驗即可）。

### 2.3.1b 延期年金保費的稅務扣除 (QDAP)

#### 概述
- 2019/20課稅年度起生效（2019年4月1日或以後繳付的保費）
- 納稅人可就承保自身及/或配偶的**合資格延期年金(QDAP)**保費享有薪俸稅和個人入息課稅的稅務扣除
- 同時適用於強積金可扣稅自願性供款

#### 稅務扣除數字

| 項目 | 數字 |
|------|------|
| **每人每年扣除上限** | $60,000（QDAP保費+強積金可扣稅自願性供款**合計**） |
| **夫婦合計** | $120,000（須每名納稅人不超過個人上限$60,000） |
| **最高節稅** | $10,200/年（按最高稅率17%計算） |

⚠️ **易考點**：$60,000的上限是QDAP保費和強積金可扣稅自願性供款**合計**的金額，不是各$60,000。不論是全做強積金、全買QDAP、還是兩者結合，上限都是$60,000。

#### QDAP認證準則（指引19）

| 準則 | 要求 |
|------|------|
| **最低保費總額** | ≥$180,000 |
| **最短保費繳付期** | ≥5年 |
| **最短年金期** | ≥10年 |
| **年金支付最低頻率** | ≥每年一次 |
| **分年結算最早日期** | 年金領取人年滿**50歲**開始 |
| **保單貨幣** | 無限制（但須披露匯率風險） |
| **無固定年期保單IRR假設** | 採用**30年**作為年金期 |

❌ **易錯點**：
- QDAP的年金最早開始年齡是**50歲**（不是65歲——65歲是香港年金計劃的認購年齡）
- 最低保費總額$180,000、最短繳付期5年、最短年金期10年——這三個數字常被混為一談

#### QDAP披露要求

- **內部回報率(IRR)**：必須在產品小冊子中以兩種形式披露——(1)保證部分及總預計利益（最低至最高IRR）；(2)以45歲非吸煙男性為例說明
- 銷售時須在利益說明文件中披露**個人化**的保證IRR和總IRR（2020年3月31日起強制）
- **保證年金款項**須佔預計年金總額的最低百分比（指引表格規定）
- **附加保障（附約）保費**不構成QDAP保費，不獲稅務扣除，須分開顯示
- 風險披露：提早退保引致重大財務損失的風險須在產品小冊子中**顯眼**披露

⚠️ **易考點**：保監局認證**不代表**保費自動獲得稅務扣除——還須符合其他稅務相關準則（如保單持有人的個人情況）。如有稅務查詢應參考稅務局網站。

#### QDAP年度摘要
- 保險人須在課稅年度（3月31日）結束後**40天內**向保單持有人發出年度摘要

#### QDAP標誌
- 必須以顯眼方式顯示在所有QDAP產品手冊上
- **嚴禁**用於任何其他保險產品或一般企業品牌營銷

### 2.3.2 退休金 (Pensions)

- 香港退休金通常被視為只惠及政府機構（如公務員）
- 非政府機構多採用**公積金計劃**(Provident Fund Schemes)，退休時一次過支付利益
- 自**2000年12月**開始實行的**強制性公積金制度**(MPF)正在產生深遠影響

### 第1-2章答題技巧

1. **可保權益只在合約開始時需要** → 離婚、還債後保單仍有效（高頻）
2. **披露責任與索償原因無關** → 未披露的疾病與索償疾病無關也可拒賠（高頻）
3. **體檢不免除披露責任** → 即使做了體檢，投保人仍須主動披露病史
4. **壽險是利益保單** → 彌償/分擔/代位權均不適用
5. **定期壽險無現金價值** → 不能作廢條款、保單抵押貸款等不適用
6. **年金核保與壽險相反** → 以生存率為基礎，年齡越高年金越高，男性高於女性
7. **抵押贖回保險是利益保險，信用壽險是彌償保險** → 債項清還後前者仍賠，後者不賠
8. **可續保/可轉換的共同特點** → 毋須健康證明，但有年齡/次數限制
9. **QDAP扣稅上限$60,000是與強積金合計** → 不是各$60,000
10. **香港年金保證期105%** → 不是100%，記住這個特殊數字

# 第3章 保險利益附約及其他產品 (24%)

## 3.1 殘疾保險

### 3.1.1 殘疾豁免保費（Waiver of Premium, WP）

**核心概念**：受保人完全傷殘後，保費由保險公司代繳，保單繼續有效。這不是「退還保費」，而是「未來保費不用再交」。

⚠️ **易考點：「等候期」的定義**
- 等候期（waiting period / elimination period）= 該附約**還未適用於有關保險單之前的一段時期**
- 也就是說，殘疾發生後，**須經過等候期（通常3或6個月）**，保費才開始被豁免
- 等候期內的保費仍須繳付，不獲豁免

❌ **易錯點：等候期 vs 免賠期**
- **等候期**（waiting period）：殘疾開始 → 等候期結束 → 開始豁免保費。是「等待豁免開始」的過渡期
- **免賠期**（deductible period）：殘疾收入保險中類似自負額的概念，免賠期內不獲收入補償
- 考題常把「等候期」混淆為「保費被豁免的時期」或「允許繳付保費的限期」——兩者皆錯

**「完全傷殘」的定義**：
- 無法執行其本職工作 → **或** 無法執行基於其教育、培訓或經驗所適合的任何其他工作
- 注意是「或」不是「及」：兩個條件中滿足任一即算完全傷殘（先看本職，再看其他適合工作）
- 年齡限制：一般15-65歲

⚠️ **易考點：豁免範圍**
- 豁免**所有保費**（包括主保單及附約保費），直至受保人康復或死亡為止
- 如果保單帶有保證可保選擇（GI），受保人在行使選擇權時已殘疾 → **自動獲得增額保障**且保費一併豁免

### 3.1.2 殘疾收入保險（Disability Income）

| 對比項 | 殘疾豁免保費（WP） | 殘疾收入（DI） |
|--------|-------------------|---------------|
| 功能 | 代繳保費 | 提供定期收入 |
| 等候期 | 3或6個月 | 1-6個月 |
| 給付形式 | 豁免未來保費 | 定期支付收入 |
| 通脹調整 | 無 | 可加COLA附約 |

**COLA（生活指數調整利益）**：殘疾收入可隨認可獨立指數（如綜合消費物價指數）掛鈎定期增加，以對抗通脹侵蝕。

## 3.2 意外死亡及喪失肢體（AD&D）

### 3.2.1 AD&D附約的核心邏輯

⚠️ **易考點：「雙倍賠償」(Double Indemnity) 是不正確的名稱**
- 原因：人壽保險單通常**不受彌償 (Indemnity) 限制**（人壽是利益保單，不是彌償保單）
- ADB提供的不是「賠雙倍」，而是「**額外**給付一筆等同於保額的款項」
- 總赔付 = 主保單基本保額 + ADB額外保額 = 2×基本保額

❌ **易錯點：ADB不是「本身賠雙倍」**
- 主保單照常賠基本保額（任何原因死亡都賠）
- ADB附約**額外**再賠一筆（僅限意外死亡時）
- 兩者是獨立的給付，不是主保單本身翻倍

### AD&D給付結構表

| 情況 | ADB附約額外赔付 | 主保單赔付 | 總赔付 |
|------|----------------|-----------|--------|
| 任何原因死亡（非意外） | 不赔 | 基本保額 | 基本保額 |
| 意外死亡 | **額外**基本保額 | 基本保額 | **2×基本保額** |
| 喪失兩肢或雙目失明 | 全額（=基本保額） | — | 全額 |
| 喪失一肢或單目失明 | 規定比率（低於全額） | — | 該比率金額 |
| 公共交通意外/公共場所火災 | ADB利益**再翻倍** | 基本保額 | **3×或4×基本保額** |

⚠️ **易考點：喪失肢體的定義和比例**
- 「喪失肢體」可解釋為**實際失去一肢體**（截肢）**或失去其功能**（永久癱瘓）
- 喪失**一肢** → 支付**規定比率**（低於全額），不是全額
- 喪失**兩肢** → 支付**全額**（等同死亡保險金）
- 喪失肢體利益**也適用於意外事故引致的失明**

❌ **易錯點：喪失一肢≠全額赔付**
- 題目問「喪失一肢支付相當於死亡保險金的款項」→ **錯**
- 只有喪失**兩肢**（或雙目失明）才支付全額
- 喪失一肢只支付規定比率

### AD&D其他關鍵細節

⚠️ **易考點：時間限制**
- 意外死亡必須於受傷後**1年內**死亡，ADB才賠付
- 超過1年後死亡 → 視為非意外死亡，ADB不賠（主保單仍賠基本保額）

**其他意外保障**：
- 每周津貼：最長**52周**
- 住院津貼：上限**1,000日**
- 加倍賠償：公共交通工具意外或公共場所火災 → ADB利益**再翻倍**（住院津貼除外）

### 近因（Proximate Cause）對AD&D的重要性

⚠️ **易考點：近因在AD&D中的角色**
- 基本壽險：大部分索償中死亡原因**無關緊要**（任何人死都賠）
- AD&D附約：死亡原因**至關重要**——必須是**意外**導致死亡才獲額外赔付
- 近因原則在此發揮作用：必須判斷意外是否為死亡的**有效主導原因**
- 如果意外受傷後因疾病死亡（疾病是近因），ADB可能不賠

## 3.3 提前支付死亡保險利益

### 3.3.1 危疾（Critical Illness）

⚠️ **易考點：危疾的三種給付條件**
1. **確診指明疾病**（如癌症、心臟病、中風等）
2. **終末疾病**（end-of-life / terminal illness）：受保人被診斷壽命預期**≤12個月**
3. **指明醫療程序**（如重大器官移植、冠狀動脈搭橋手術）

❌ **易錯點：危疾受保事件**
- 危疾附約通常包括的受保事件：**傷殘**、**免疫系統相關疾病**、**癌症**
- **感冒**不包括 → 考題常以感冒作為干擾項

**危疾關鍵數字**：
- 等候期：**90天**（保單生效後首90天內確診的疾病不賠）
- 年齡上限：**80歲**
- 豁免保費：喪失工作能力**3個月**後開始豁免

### 3.3.2 長期護理（Long-term Care）

| 項目 | 內容 |
|------|------|
| 給付條件 | 須接受長期護理 |
| 等候期 | **90日** |
| 護理院每月利益 | 死亡保險金**2%** |
| 家中護理每月利益 | 死亡保險金**1%** |
| 最高支付 | 死亡保險金**50%-100%** |
| 保單須已生效 | **1年以上** |
| 豁免保費 | 接受護理期間豁免 |

⚠️ **易考點：提前支付的後果**
- 受益人須簽署**棄權聲明**（waiver），同意提前支付數額將從將來的死亡保險金中扣除
- 這不是「額外送你的錢」，而是「提前預支你死後的錢」

### 3.3.3 危疾 vs 長期護理 對比

| 對比項 | 危疾 | 長期護理 |
|--------|------|---------|
| 觸發條件 | 確診指明疾病/終末疾病/醫療程序 | 須接受長期護理 |
| 等候期 | 90天 | 90日 |
| 給付方式 | 一次性 | 按月（2%/1%） |
| 年齡上限 | 80歲 | — |
| 保單須已生效 | — | 1年以上 |
| 豁免保費 | 傷殘3個月後 | 護理期間 |

## 3.5 保證可保選擇（Guaranteed Insurability, GI）

**含義**：在指定日期/年齡/事件發生時，**毋須可保性證明**即可購買增額保險。

⚠️ **易考點：GI的關鍵限制**
- 增額保險金額受限（限於原有保單面額或更少）
- 通常限**40歲前**行使
- **非自動獲得**：錯過已觸發的選擇權 → 喪失該次權利，但可在下輪行使
- 指定事件：結婚、子女出生等

**與WP附約的互動**：若保單已有WP附約，受保人行使GI選擇權時已殘疾 → **自動獲得增額保障**，保費一併豁免。

# 第4章 闡釋人壽保險單 (24%)

## 4.1 完整合約條款（Entire Contract Provision）

人壽保險在索償時**必須出示保單正本**（與一般保險不同），因此完整合約條款至關重要。

**四項規定**：
1. 完整合約 = 保險單 + 附約 + 投保單複印本（**封閉式合約**）
2. 只有保險公司**指定高級職員**有權更改合約
3. 除非**書面形式**，否則修改不生效
4. 除非得到**保單所有人同意**，否則不能修改

⚠️ **易考點：封閉式合約的意義**
- 投保單複印本附於保單中 → 保險人不能引用投保單以外的口頭承諾或陳述
- 保護雙方：保險人不受代理人越權承諾約束；保單所有人不受保險人單方修改影響

---

## 4.2 不可異議條款（Incontestability Provision）

### 核心規則

⚠️ **易考點：不可異議的三個條件**
1. 保單發出日起計已生效**2年**
2. **受保生命在世時**（強調「在世」——見下方解釋）
3. **欺詐例外**（香港法律不助長欺詐行為）

**為什麼要強調「受保生命在世時」？**
- 如果省略此詞組，受益人可能在受保人於可異議期內死後，**故意延遲提出索償申請**，等到2年可異議期結束後才申請，以圖取得此條文保護
- 加入「在世」條件 → 受保人必須在2年可異議期內仍然生存，條文才適用

### 欺詐例外

⚠️ **易考點：欺詐不受不可異議條款保護**
- 根據香港法律，若證實索償人或被保險人曾作**欺詐行為**，便不能依賴不可異議條款
- **不論合約內容如何**，香港法律不會助長欺詐行為
- 區別：一般的不披露/失實陳述（非欺詐性）→ 2年後不可異議；欺詐性違反 → **永遠可異議**

### 不可異議條款不能防止的事

❌ **易錯點：不可異議條款的適用範圍**
- **可以**防止：保險人以違反最高誠信（不披露/失實陳述）為由質疑合約有效性
- **不能**防止：保險人提出合約**非法**（如缺乏可保權益 → 合約自始無效）
- **不能**防止：欺詐行為

### 個案六要點（不可異議條款案例）

**案情**：保單所有人在簽署投保申請書後第4天下午才求診並被診斷患鼻咽癌，投保時未披露此病徵。保單生效3年後死於鼻咽癌。

**裁決要點**（索償人得直，兩個獨立理由，任一足夠）：
1. **未違反最高誠信**：投保人在簽署投保申請書時尚不知道自己患癌（病徵類似傷風感冒，非醫學專家）；且保險合約在簽署投保申請書時（非保單簽發時）訂立，披露責任在合約訂立時終止
2. **不可異議條款**：保單生效已逾2年，且無欺詐證據 → 保險人不能撤銷合約

⚠️ **易考點：英格蘭法院判決**
- 英格蘭法院曾判：不可爭議條款**不能防止**保險人以被保險人**疏忽的失實陳述**為由拒賠
- 除非該條款**明文述及疏忽**，或除非不這樣理解條款便不能理解
- 這是英式法例的特點，與香港的欺詐例外不同

## 4.3 寬限期（Grace Period）

### 基本規則

| 項目 | 內容 |
|------|------|
| 期限 | **30或31日** |
| 不適用於 | **首期保費** |
| 寬限期內繳付 | 視作**準時繳付** |
| 性質 | **並非免費保險期** |

⚠️ **易考點：寬限期不是免費保險期**
- 寬限期內死亡 → 保險人會從死亡保險金中**扣除到期保費**
- 不是「白送你一個月保障」，而是「容許你遲交保費，但欠的保費仍要付」

### 英式 vs 美式保單的關鍵區別

❌ **易錯點：失效日期不同**

| | 英式保單 | 美式保單 |
|---|---------|---------|
| 失效日期 | **保費到期日**開始失效 | **寬限期屆滿時**失效 |
| 效果 | 追溯至到期日失效 | 多送一個月保障 |
| 實際差異 | 寬限期內死亡仍須扣保費 | 寬限期內死亡仍須扣保費 |

- 兩者在寬限期內死亡時的處理相同（都從保險金扣保費）
- 區別在於：寬限期結束時受保人仍活著但未交保費 → 英式追溯到到期日失效；美式在寬限期滿時才失效（等於多送一個月保障）

## 4.4 受益人的指定（Beneficiary Designation）

### 受益人類型

| | 可撤換受益人（Revocable） | 不可撤換受益人（Irrevocable） |
|---|--------------------------|----------------------------|
| 變更條件 | 保單所有人可隨時變更 | 須取得受益人**書面同意**才能變更 |
| 權利性質 | 僅有**期望**（expectation），非既得權利 | **既得權利**（vested right） |
| 轉讓限制 | 不構成轉讓障礙 | 轉讓不得侵犯其既得權利 |

### 受益人指定方式

⚠️ **易考點：概括式指定（Class Designation）**
- 可以指定一組人而非指名每個人，例如「我的子女」「我的兄弟姊妹」
- 第一順位受益人（primary beneficiary）：死亡保險金應付時收取
- 次順位受益人（contingent beneficiary）：所有第一順位受益人都不能比受保生命活得久時收取
- 多名受益人 → 每人所得金額**相等**（除非保單另外指明）

### 《已婚者地位條例》與法定信託

⚠️ **易考點：為配偶/子女買壽險 = 法定信託**

**法律效果**：
- 保單持有人為配偶及/或子女的利益而購買壽險 → 達到建立**法定信託**的效用
- 配偶及/或子女成為保單的**實益擁有人**（beneficial owner）
- 保單所有人成為**受託人**（trustee）

**關鍵後果**：
- 此等實益權益被視為**連贈與人本人也無法收回的饋贈**（「生者之間的贈予」）
- 此等權益現已屬於受益人的**各別財產**（Estate）
- 即使保單措辭容許撤換受益人，**衡平法不會容許**原受益人的既得實益權益受損

❌ **易錯點：可撤換 ≠ 真的能撤換**
- 即使保單條款容許撤換受益人，如果適用《已婚者地位條例》的法定信託 → 衡平法保護下，實際上無法損害原受益人權益
- 考題常測試：保單所有人能否在配偶不知情下將受益人改為其他人 → 答案是不能（法定信託保護）

### 保險人的多重申索風險

保險人可能收到多個相互衝突的申索，申索人可能是：
- 保單受益人、承讓人、保單受託人、信託受益人、破產管理人、遺產代理人

⚠️ 保險人如果理所當然地認為受益人指定條文具有最高效力 → 將面對**須作雙倍給付的風險**

## 4.5 不能作廢條款（Nonforfeiture Benefits）

### 核心概念

「不能作廢」= 在中斷繳付保費的情況下，保單仍然有效。

⚠️ **易考點：適用範圍**
- 適用於大多數傳統人壽保險計劃
- **定期壽險除外**（定期壽險無現金價值，不適用不能作廢條款）

### 保費中斷時的處理邏輯

**第一步：自動保費貸款條款（APL）觸發**
- 如果保單所有人只是中斷繳費，**沒有發出選擇通知** → 自動觸發APL
- APL：現金價值被用作支付到期保費，直至用完為止，保持保單全額有效
- 部分保險人視APL為獨立條款而非不能作廢條款

**第二步：保單所有人主動選擇（三選一）**

| 選擇 | 內容 | 保額 | 保單性質 |
|------|------|------|---------|
| **退保現金價值** | 中止保單，獲付退保價值 | — | 保單終止 |
| **減額清繳保險** | 以淨現金價值作整付保費 | **較低**（同計劃） | 永久有效 |
| **展期保險** | 以淨現金價值作整付保費 | **相同** | 定期壽險，期限取決於現金價值 |

⚠️ **易考點：默認選擇**
- 如果保單**不包含**APL條款，且保單所有人未作出選擇 → 自動採用**展期保險**（(b)(iii)項）
- 如果保單**包含**APL條款，且保單所有人未作出選擇 → 自動觸發**APL**

❌ **易錯點：減額清繳 vs 展期保險**
- 減額清繳：**保額降低**，但**計劃相同**（如終身壽險仍是終身壽險），永久有效
- 展期保險：**保額不變**，但變成**定期壽險**（不再是原計劃），期限有限
- 記憶法：「減額清繳 = 保額減、計劃不變；展期 = 保額不變、計劃變」

### 三個選擇對比表

| 對比項 | 退保現金價值 | 減額清繳保險 | 展期保險 |
|--------|-------------|-------------|---------|
| 保單是否繼續 | 否（終止） | 是 | 是（但變成定期壽險） |
| 保額 | 不適用 | 降低 | 不變 |
| 計劃類型 | 不適用 | 與原保單相同 | 定期壽險 |
| 資金來源 | — | 淨現金價值作整付保費 | 淨現金價值作整付保費 |
| 期限 | — | 永久 | 取決於現金價值多少 |

## 4.6 保單抵押貸款（Policy Loan）

直接源自保單現金價值的特點：

| 項目 | 內容 |
|------|------|
| 貸款上限 | 現金價值（扣除一年貸款利息） |
| 抵押 | 唯一抵押 = 保單現金價值 |
| 用途 | 無任何限制 |
| 還款 | 自行決定多少和何時還 |
| 利率 | 訂明或設上限 |
| 未償還貸款 | 從死亡保險金或退保價值中扣除 |

## 4.7 復效（Reinstatement）

### 核心概念

復效 = 已失效（已死）的保單重獲「生命」。英式又稱「保單復效」（Policy Revival）。

⚠️ **易考點：復效只適用於已失效，不適用於已退保**
- **已失效（lapsed）**的保單 → 可以復效
- **已退保（surrendered）**的保單 → **不可以復效**
- 這是考試高頻考點

### 復效條件

| 條件 | 內容 |
|------|------|
| 申請期限 | 一般**5年**內（因保險公司而異） |
| 可保性證明 | 須提供持續可保性（健康狀況良好）證明 |
| 未繳貸款 | 繳付任何未繳清的貸款連利息 |
| 未繳保費 | 補付已到期但未繳清的保費及利息 |
| 復效費 | 繳付一筆復效費 |

⚠️ **易考點：復效後重新啟動兩個期間**
- 從復效日計起的**另一段可異議期**（新2年可異議期）
- 從復效日計起的**另一段自殺免責期**（新1年自殺免責期）
- 這意味着復效後的保單在這兩方面等同於新保單

❌ **易錯點：復效 vs 新保單**
- 復效不是買一份新保單，而是讓原保單重新生效
- 但復效後會產生新的可異議期和自殺免責期（對保險人的保護）
- 原保單的現金價值、已繳保費年限等可能保留（視保險人安排）

## 4.8 誤報年齡或性別（Misstatement of Age/Sex）

### 索償後發現錯誤

⚠️ **易考點：英式做法只能向下調整**

| 情況 | 美式做法 | 英式做法 |
|------|---------|---------|
| 誤報導致保費過低（應繳更多） | 向上調整利益金額（降低赔付） | 向下調整利益金額（降低赔付） |
| 誤報導致保費過高（應繳更少） | 向上調整利益金額（提高赔付） | **只退還超額保費**（不含利息），**不向上調整**利益 |

❌ **易錯點：英式做法**
- 英式做法：任何對保險金額的調整**只能是向下的**
- 如果多繳了保費 → 退還超額保費（**不包括利息**），但不提高赔付金額
- 必須與採用英式保單的保險人查清楚

### 索償前發現錯誤

保單所有人可以選擇：
1. **保持保額不變** → 計算正確保費後，收取退費或支付額外保費
2. **調整保單保額** → 新保額 = 已交保費本可按正確年齡/性別購得的金額

註：索償前發現錯誤時，英式慣例與美式相同（均可調整）。

## 4.9 轉讓（Assignment）

### 法律基礎

《法律修訂及改革（綜合）條例》第9條容許藉訂明手續轉讓據法權產，而對保險合約的權益屬於據法權產。

⚠️ **易考點：保險合約權益的性質**
- 屬於**現時的**（present），而非未來的 → 可以轉讓
- 也可形容為**復歸的**（reversionary）→ 權利已獲承認，但尚未能實際享用，直到未來某日或某事發生後

### 轉讓的關鍵特點

**(a) 轉讓通知**
- 轉讓自向保險人發出**書面通知**那一天起生效
- 保險人在未收到書面通知前，**沒有義務**按照轉讓行事
- 此規定**無意阻止**進行轉讓（只是通知機制）

**(b) 轉讓的有效性**
- 保險人對轉讓的有效性**卸棄責任**
- 轉讓人應就有效轉讓所需手續尋求**獨立法律意見**

**(c) 承讓人的權利**

⚠️ **易考點：承讓人收取的是淨保單收益**
- 承讓人繼承轉讓人的一切權利和補救
- 但承讓人收取的**不可能比轉讓人可收的多**
- 如果轉讓人曾藉欺詐或失實陳述購得保險 → 保險人可向承讓人提出免責辯護
- 保險人可向承讓人執行**抵銷權**：轉讓人任何未付過期保費和未償還保單抵押貸款連利息 → 從應向承讓人支付的保單利益中扣除
- 結果：承讓人收取的是**淨保單收益**（net policy proceeds）

**(d) 轉讓利益而非負擔**
- 法律不容許一人轉讓其欠第三人的義務（如支付保費的義務），除非得到第三人同意

**(e) 轉讓限制**
1. 不得侵犯任何受益人（尤其**不可撤換受益人**）的既得權利
   - 注意：指定**可撤換受益人**只能帶來**期望**，而非既得權利
2. 不能為非法目的（如洗黑錢）
3. 可能規定只能以整筆保單利益支付承讓人（不能選其他賠付選擇）

### 絕對轉讓 vs 抵押轉讓

| 對比項 | 絕對轉讓（Absolute Assignment） | 抵押轉讓（Collateral Assignment） |
|--------|-------------------------------|--------------------------------|
| 性質 | **不可撤銷** | **暫時性** |
| 轉讓內容 | 壽險單的**一切擁有權** | 限於貸款金額連利息 |
| 用途 | 執行合約或饋贈 | 以保單作為非保險人貸款的質押擔保 |
| 還款後 | — | 轉讓人有**復歸權**，取回權益 |
| 保單抵押貸款 | — | 已通知的抵押轉讓生效後，轉讓人**不能**獲取保單抵押貸款 |
| 退保 | — | 已通知的抵押轉讓生效後，轉讓人**不能**退保 |

⚠️ **易考點：抵押轉讓生效後的限制**
- 已發通知的抵押轉讓生效之時 → 轉讓人不能獲取保單抵押貸款或退保
- 這是保護承讓人（貸款人）利益的措施

## 4.10 紅利選擇（Dividend Options）

### 三種紅利類型

| 紅利類型 | 給付時間 | 性質 |
|---------|---------|------|
| 現金紅利 | 每年 | 即時應付給保單所有人 |
| 復歸紅利 | 每年 | 英式紅利 |
| 期終額外紅利 | 期滿 | 期末給付 |

### 現金紅利的五種選擇

1. 立即以**現金**支付
2. 用以**繳付未來保費**
3. **存放在保險人處**賺取利息（紅利存款與現金價值是分開的）
4. 購買**清繳增額保險**（同樣會產生紅利）
5. 購買**為期一年的定期壽險**

⚠️ **易考點：自動紅利選擇**
- 如果保單所有人未作出選擇 → 採用「自動紅利選擇」
- 香港常見做法因保險人而異，較常見：
  - 存放在保險人處賺取利息（選項c）
  - 購買清繳增額保險（選項d）

## 4.11 賠付選擇（Settlement Options）

| 選擇 | 內容 | 本質 |
|------|------|------|
| 整筆支付 | 一次性付款 | 完成整份保險合約 |
| 利息選擇 | 收益存放於保險人處，定期支付利息 | 本金不動，只付利息 |
| 定期支付選擇 | 約定期間內分期支付等額收益及利息 | 等同購買確定年金 |
| 定額支付選擇 | 分期給付約定等額，直至付完 | 本息耗盡為止 |
| 終身收益選擇 | 收款人在世時分期支付 | 等同購買終身年金 |

⚠️ **易考點：終身收益選擇的預期付款額**
- 與定期支付或定額支付相比，終身收益選擇的**預期分期付款額會較低**
- 原因：終身年金需考慮收款人壽命不確定性，保險人承擔長壽風險，故每期支付較少

## 4.12 自殺除外責任（Suicide Exclusion）

### 核心邏輯

**為什麼人壽保險最終會賠自殺？**
- 人壽保險原意是為受贍養人提供保障，而非讓受保人本身得益
- 長期合約中，因受保人自殺而懲罰其家人是不公平的
- 但為防止有人藉自殺取得保險金，保險人設免責期是合理的

### 自殺條款規則

| 項目 | 內容 |
|------|------|
| 免責期 | 通常**1年**（保單發出後） |
| 免責期後自殺 | **如常支付**死亡保險金 |
| 免責期內自殺 | **不賠**，但退還已付保費（減去未繳貸款及利息） |

⚠️ **易考點：舉證責任在保險人**
- 自殺列為除外責任 → 證明死亡是由自殺引起的**舉證責任落在保險人身上**
- 這往往並非容易的事
- 不同於一般索償（受益人只需證明死亡），自殺免責需保險人主動舉證

⚠️ **易考點：免責期內自殺的退費**
- 免責期內自殺 → 退還**已付保費**（減去任何未繳清的貸款及利息）
- 注意：退的是「已付保費」，不是「保額」

❌ **易錯點：免責期內自殺 ≠ 不退任何錢**
- 有些考生以為免責期內自殺 = 什麼都不賠、什麼都不退
- 實際上：不賠死亡保險金，但**退還已付保費**（扣除欠款）

### 通融賠付（Ex Gratia Payment）
- 曾有案例：有理由假設已證實的自殺是由保單生效後的事件引起的（否則屬於除外責任），保險人仍作出賠償
- 這僅屬**通融賠付**（法律上並非必要），情況可能頗特殊

## 甲類題目

**Q1：殘疾豁免保費附約中的「等候期」是指？**
- (a) 保費被豁免的時期 → 錯
- (b) 允許保單所有人繳付保費的限期 → 錯
- (c) 該附約還未適用於有關保險單之前的一段時期 → **正確**
- (d) 殘疾時保費被豁免前的一段時期 → 錯（這看似正確但混淆了概念）

**Q2：「雙倍賠償」(Double Indemnity) 是不正確的名稱，因為？**
- (a) 人壽保險單通常不受彌償限制 → 正確但不是最佳答案
- (b) 支付的數額並不一定是保額的雙倍 → **正確**（喪失肢體時支付規定比率，非雙倍）
- (c) 它僅在因意外導致死亡的情況下給付 → 這是事實但不是「名稱不正確」的原因
- (d) 受益人因為同一事件收取兩次利益付款是違法的 → 錯

## 乙類題目

**Q3：AD&D附約的正確評論？**
- (i) 喪失肢體可解釋為實際失去一肢體，或失去其功能 → **正確**
- (ii) 喪失一肢，則支付相當於死亡保險金的款項 → **錯**（喪失一肢只支付規定比率）
- (iii) 喪失兩肢，則支付相當於死亡保險金的款項 → **正確**
- (iv) 喪失肢體利益也適用於意外事故引致的失明 → **正確**
- 答案：**(i)、(iii)及(iv)**

**Q4：危疾附約通常包括的受保事件？**
- (i) 傷殘 → **正確**
- (ii) 免疫系統相關疾病 → **正確**
- (iii) 感冒 → **錯**
- (iv) 癌症 → **正確**
- 答案：**(i)、(ii)及(iv)**

| 寬限期=免費保險期 | 錯。死亡會從保險金**扣除到期保費** |

| 英式誤報年齡可向上調整赔付 | 錯。英式做法**只能向下調整**，多繳保費退還（不含利息） |
| 不可異議條款可防止合約非法抗辯 | 錯。不可異議**不能**防止合約非法（如缺乏可保權益） |
| 免責期內自殺不退任何錢 | 錯。退還**已付保費**（減去未繳貸款及利息） |
| 可撤換受益人=可以隨意撤換 | 不完全。如適用《已婚者地位條例》法定信託→衡平法保護下實際無法損害原受益人權益 |
| 轉讓需保險人同意才有效 | 錯。轉讓自向保險人發出**書面通知**起生效，不需保險人同意 |
| 承讓人可收取比轉讓人更多 | 錯。承讓人收取的**不可能比轉讓人可收的多**，保險人可行使抵銷權 |

## 5.2 投保

### 5.2.1 投保手續

投保單是核保的主要（有時是唯一）資料來源，填寫時須注意：

- **重要事實**必須全部提供；健康問題答「是」須附詳細解釋及日期
- 投保人通常須**親自填寫**；如由中介人代填，須確保客戶知悉這是他的聲明
- 改動及修正須由投保人簽名並註明日期
- 須清楚註明**生效日期**（保險人通常允許回溯一段時期）
- 須驗證投保人及受保生命的**身份**
- **下一個生日年齡**（Age next birthday）影響保費

**補充要求**：壽險代理人報告（購買原因、認識客戶多久）、繳費方式、可保性證明、核保問卷。

### 5.2.2 收據及保單生效

⚠️ **核心原則**：人壽保險保單一旦開始生效，保險人**不能取消**。這與非人壽保險不同（非壽險一般有條款容許取消）。

保費收據是承認保險人收到投保人提交投保單並繳付首期保費的書面證明文件。共有兩種：

#### 對比：附條件保費收據 vs 立約保費收據（TIA）

| 比較項目 | 附條件保費收據 | 立約保費收據（TIA） |
|----------|---------------|-------------------|
| **又稱** | Conditional premium receipt | 臨時保險協議(TIA) / 不附條件保費收據 |
| **生效時點** | 投保時即生效，**但須事後確定可保性** | 投保人簽署投保單並繳清保費之日即生效 |
| **生效條件** | 須事後確定投保人按**標準條款**具有可保性 | **不以**事後可保性證明為條件 |
| **性質** | 有條件 | **獨立合約**，獨立於其後簽發的保險保單 |
| **保障期限** | 60/90天 | 通常30天（指定天數） |
| **可否提前終止** | — | 可以：(1)保險人退還保費；(2)寄出終止通知書後指定天數；(3)已簽發保單生效 |
| **若投保後變得不可保** | 只要確定**投保當時**是可保的，仍獲保障 | 保障已生效，不取決於事後可保性 |
| **若可保性只適用另一計劃** | 原合約未生效（要約未按原訂條款獲接受） | — |

⚠️ **易考點**：
- 附條件收據的關鍵：投保**當時**可保即可，即使投保後變得不可保甚至死亡，仍獲保障
- TIA 是一份**獨立合約**，不是保單的預備文件
- TIA 保障期最長60或90天，可以在保單生效前提前終止

❌ **易錯點**：非人壽保險中類似的暫保單（Cover Note）通常只保障30天，且未必受是否繳付保費影響。不要把暫保單與TIA混淆。

#### 附條件收據的兩個特殊情況

1. **可保性只適用另一計劃**：如果確定投保人的可保性只適用於不同保費或不同保額的計劃，原合約**不是**由投保當日生效（要約未按原訂條款獲接受），須雙方達成修訂條款後才生效
2. **投保後不可保或死亡**：只要確定投保**當時**是可保的，仍會獲保障

### 5.2.4 冷靜期（指引29）

**背景**：公眾擔心人壽保險推銷員在銷售時可能太過急進甚至帶有壓迫感，人們可能受壓力下購買不想要或負擔不起的保險。指引29由保監局依據《保險業條例》第133條制訂。

#### (a) 適用範圍

指引29適用於所有經營長期業務的獲授權保險人及所有就長期業務進行受規管活動的持牌保險中介人。

**適用於所有新訂立的人壽保險保單，但以下除外**：

| 不適用情況 | 原因 |
|-----------|------|
| 團體保單 | 團體保險有集體核保，非個人推銷 |
| 類別G、H、I保單 | 資本贖回、永久健康等特殊類別 |

**以下交易也不適用**：

| 不適用交易 | 原因 |
|-----------|------|
| 提高現有保單保額（增加保費） | 非新訂合約 |
| 行使抗通脹條文提高保額 | 保單條款及條件下的權利 |
| 為現有保單加入新附加保障 | 非新保單 |
| 行使轉換權（如定期轉終身） | 現有保單條款下的權利 |

⚠️ **易考點**：冷靜期不適用的場景與轉保不適用的場景有部分重疊（團體保單），但冷靜期還多了G/H/I類別和提高保額/轉換權等。

#### (b) 冷靜期計算

| 項目 | 規定 |
|------|------|
| **冷靜期長度** | **21個曆日** |
| **起算日** | 保單或冷靜期通知書交付予保單持有人（或指定代表）之日後的**翌日**起計 |
| **以較早者為準** | 保單交付日 vs 冷靜期通知書交付日 |
| **交付當天不算** | 交付當天不包含在21個曆日內 |
| **最後一天非工作天** | 冷靜期延長至隨後的**第一個工作天** |

⚠️ **易考點**：「曆日」不是「工作日」！21個曆日≈3週。如果第21天是星期日或公眾假期，則順延至下一個工作天。

#### (c) 保單交付限期（9個曆日限期）

| 交付方式 | 規定 |
|---------|------|
| 保險人直接交付 | 保單簽發日期起計**9個曆日**內交付 |
| 透過中介人交付 | 保險人須在9個曆日限期結束前預留充足時間提供保單給中介人；中介人須盡合理努力在9個曆日內交付 |
| 冷靜期通知書 | 保險人應在9個曆日限期內直接交付 |
| 最後一天非工作天 | 限期延長至隨後的第一個工作天 |

❌ **易錯點**：9個曆日是保單**交付**限期，不是冷靜期本身。冷靜期是21個曆日。9個曆日是確保保險人不能無限期拖延交付保單，從而推遲冷靜期的開始。

#### (d) 冷靜期內取消保單的權利及退費

| 保單類型 | 退費規則 |
|---------|---------|
| 一般保單（除類別C及整付保費外） | 退還**全數（100%）已繳保費** |
| 類別C保單（投連壽險） | 保險人有權作**市值調整** |
| 整付保費保單 | 保險人有權作**市值調整** |

**市值調整的限制**：
- 只可參照變現投資時可能出現的虧損
- **不得**將簽發保單的費用或佣金支出計算在內
- 須在產品小冊子中披露市值調整權利及計算基準
- 準保單持有人簽署投保申請書前須獲告知

⚠️ **易考點**：退費規則的邏輯——一般保單全額退費是因為沒有投資波動風險；投連壽險和整付保費有投資風險，所以允許市值調整，但調整只能反映投資虧損，不能扣除行政費用。

#### (e) 取消方式

- 保單持有人須**直接向保險人**提供**書面通知**要求取消並退回保單（如適用）
- 可在冷靜期內**任何時間**行使

#### (f) 保險人及中介人的義務

| 義務人 | 義務內容 |
|--------|---------|
| 保險人 | 交付冷靜期通知書（含保單可供領取情況、冷靜期屆滿日期、退費權利、客戶服務聯絡資料） |
| 保險人 | 備存證據紀錄（如確認交付的收據副本） |
| 中介人 | 簽署投保申請書前告知準保單持有人冷靜期權利、屆滿日期、市值調整權利（如適用） |
| 保險人 | 投保申請書內緊接簽署欄正上方印上冷靜期陳述（字體不得小於其他聲明） |
| 保險人 | 交付保單時附上冷靜期提示 |

### 5.2.5 轉保（指引27）

**背景**：人壽保險是長期合約，設計宗旨為保障人的壽命，通常涉及較長年期。保單持有人可能在期後考慮購買另一份保單以取代最初購買的保單。此時應注意更改最初保單的不利之處（如退保收費）。

#### (a) 適用範圍

指引27適用於所有經營長期業務的獲授權保險人及持牌保險中介人。

**不適用的情況**：

| 不適用情況 | 條件 |
|-----------|------|
| 團體保單 | — |
| 數碼渠道直接銷售 | 須同時滿足4個條件（見下） |
| 保監局事前書面同意豁免 | 如保險科技沙盒 |

**數碼渠道豁免的4個條件（須全部符合）**：
1. 由保險人或認可機構（以代理機構身份）直接銷售
2. 通過數碼分銷渠道（網頁或手機應用程式）
3. 銷售前或期間**未向客戶提供建議**
4. 須以顯著方式展示聲明，警告客戶如為取代現有保單應尋求專業意見

#### (b) 轉保的定義

轉保須**同時符合**以下三個條件：

1. 客戶現時或曾經擁有另一份（或多份）人壽保險保單（「現有人壽保險保單」）
2. 現有保單的保單持有人與新保單的保單持有人**相同**
3. 客戶透過以下方式使用現有保單的現金價值，或使用減少保費而節省的金額，以資助購買新保單：
   - (1) 行使權利（提取保單款項、提早退保）或保單自動運作（失效、減額繳清、展期保單）以減少現金價值或保額
   - (2) 從現有保單提取保單貸款
   - (3) 暫停或終止支付現有保單的保費

⚠️ **易考點**：轉保的核心是**用舊保單的錢買新保單**。如果只是買了一份新保單而完全不動舊保單，不構成轉保。

#### (c) 不視為轉保的情況

| 不視為轉保的情況 | 原因 |
|----------------|------|
| 根據現有保單條文**轉換**為新保單 | 這是保單權利的行使，非市場行為 |
| 同一保險人發出、**毋須再次核保**、按轉換/轉移計劃進行 | 內部產品轉換 |
| 只變更附加條款保障，**基本計劃不變** | 非整體取代 |
| 購買新保單是為了取代在**冷靜期內取消**的現有保單 | 冷靗期保障已涵蓋 |

❌ **易錯點**：不要混淆「轉換權」（如定期轉終身，屬現有保單條款權利）與「轉保」（用舊保單現金價值買新保單）。前者不視為轉保。

#### (d) 《重要資料聲明書—轉保》

當新保單被識別為轉保個案：

| 步驟 | 規定 |
|------|------|
| 簽署聲明書 | 保險人或中介人須要求客戶簽署《重要資料聲明書—轉保》 |
| 解釋內容 | 中介人須向客戶解釋轉保的影響和風險 |
| 正本保管 | 新保單的保險人保管正本 |
| 副本提供 | 保險人須在新保單簽發日期或之前向客戶提供聲明書副本 |
| 紀錄保存 | 保險人須保存已簽署聲明書及發送副本的日期和方式 |

#### (e) 內部轉保

「內部轉保」指新保單的保險人同時為客戶現有保單的保險人。

| 項目 | 規定 |
|------|------|
| 核對範圍 | 緊接新保單申請日期前**12個月**內 |
| 核對內容(1) | 客戶在12個月內是否曾與該保險人訂立現有保單 |
| 核對內容(2) | 現有保單在12個月內是否出現減少現金價值/保額、提取貸款、暫停繳費等情況 |
| 事後監察 | 如申請時不構成轉保，保險人須繼續監察申請日後**12個月**內現有保單有否變化 |

⚠️ **易考點**：內部轉保的12個月核對是雙向的——向前看12個月（申請前是否有舊保單變動）+ 向後看12個月（申請後舊保單是否變動）。

#### (f) 中介人評估義務

持牌保險中介人須為客戶評估購買新保單是否符合客戶的**最佳利益**，必須考慮：
- 減少現有保單現金價值或保額的影響
- 因轉保而對客戶造成的其他不利後果

中介人須妥善記錄：客戶提供的資料、考慮因素、評價及建議、建議理由。

### 5.2.6 長期保險保單利益說明（指引28）

**目的**：載述向潛在或現有保單持有人提供利益說明文件的標準規定，確保保單持有人獲得充足及清晰的資料。

#### (a) 適用範圍

- 適用於所有經營長期業務的獲授權保險人
- 就所有**具現金價值**的人壽保險保單而言（團體保單除外），必須遵從
- 具可退款特點的個人保單被視為具現金價值的人壽保險保單

#### (b) 三種利益說明

| 類型 | 提供時間 | 簽署要求 |
|------|---------|---------|
| **銷售時利益說明** | 簽署申請表前供審閱 | 須客戶簽署 |
| **補充利益說明** | 保單持有人選擇產品選項時（如保費假期、部分退保） | 保險人可選擇是否要求簽署 |
| **有效利益說明** | 保單發出後**至少每年一次** | **無須**客戶簽署 |

#### (c) 一般原則

- 利益說明須基於個別客戶詳情（年齡、性別、吸煙習慣），不得使用通用樣本
- 「繳付保費總額」欄須在利益款項欄**之前**顯示
- 退保發還金額須在身故賠償額**之前**顯示
- **不得重點指出**非保證數字（不得用粗體、下劃線、顏色等特別格式）
- 須使用指引28附件中可接納的保險術語

#### (d) 按保單類型的具體規定

##### (i) 投資壽險保單（指引28-附件I）

| 項目 | 規定 |
|------|------|
| 假設淨回報率（選項1） | **0%、3%、6%、9%**，以不同列表分別顯示退保發還金額及身故賠償額 |
| 假設淨回報率（選項2） | **0%、3%、6%**，在同一列表中顯示 |
| 最高回報率 | 除0%外，3%/6%/9%為可採用的**最高**回報率（可用較低的） |
| 費用及收費 | 須反映所有保單層面的費用（不包括基金管理費）；基金管理費須透過數字說明反映 |

⚠️ **易考點**：投資壽險有兩個選項，一個用4個回報率（分列表），一個用3個回報率（同列表）。9%是最高可用回報率。

##### (ii) 分紅保單（指引28-附件II）

| 項目 | 規定 |
|------|------|
| 基本情景 | 委任精算師參照精算指引9制定最佳估算假設；投資回報率**不得高於**最佳估算 |
| 悲觀情景 | 採用預計投資回報的**第25百分位數**（可用更低的，不可用更高的） |
| 樂觀情景 | 採用預計投資回報的**第75百分位數**（不可用更高的） |
| 其他假設 | 保留不變（紅利/可支取現金的累積利率除外） |
| 悲觀/樂觀顯示 | 須在**同一頁面**顯示 |
| 顯示年度 | ≥30年（第5個保單年度後至少每5年顯示）、65歲時、100歲時、保單年期屆滿時 |
| 補充/有效利益說明 | 可選擇是否顯示悲觀/樂觀情景；如選擇顯示與基本情景不同的情景，最低限度須顯示**悲觀情景** |

❌ **易錯點**：悲觀情景用第25百分位數，樂觀用第75百分位數——不要記反。悲觀=低回報=低百分位數。

##### (iii) 萬用壽險（非投資相連）（指引28-附件III）

| 項目 | 規定 |
|------|------|
| 回報率假設(1) | **保證基礎/保守假設基礎**：按保單訂明的最低保證派息率；如無保證，用0% |
| 回報率假設(2) | **現時假設基礎**：按保險人預測的現行派息率 |
| 保證基礎的費用 | 採用**最高**費用及收費標準 |
| 現時假設基礎的費用 | 採用**現時**費用及收費標準 |
| 保險成本費率 | 須披露**首10個保單年度**各年度的保險成本費率；其後可按每5年顯示 |
| 悲觀/樂觀情景 | **可選擇是否提供**；僅在(1)目標資產組合有≥20%股權投資及(2)精算師認為不會誤導時才可提供 |
| 「保證基礎」一詞 | 僅在**全數預計價值均為保證**時方可使用；否則用「保守假設基礎」 |

⚠️ **易考點**：萬用壽險的悲觀/樂觀情景有額外門檻——≥20%股權投資+精算師判斷不誤導。分紅保單則必須提供悲觀/樂觀情景。

##### (iv) 非分紅保單（指引28-附件IV）

| 項目 | 規定 |
|------|------|
| 原則 | 整個保單期內僅提供**保證利益價值** |
| 任何非保證要素 | 須遵從分紅保單的標準說明 |
| 顯示年度 | 與分紅保單相同（≥30年、65歲、100歲、屆滿） |

#### (e) 保費抵銷的特殊規定（指引16）

- 須就不同情景（尤其紅利減少而未能抵銷保費的不利情景）作出推算
- **不得使用**「消失」、「保費消失」或意指保費已繳清的類似字眼
- 須提醒客戶：能否持續以紅利抵銷保費取決於未來紅利，紅利多寡無法保證

### 5.2.7 派發保單紅利

#### 紅利的基本原理

- 紅利金額與**匯集基金**（pooled fund）的業績直接相關
- 匯集基金 = 保險人因提供分紅保單而在資產負債表上顯示的資產全部
- **投資回報**對紅利額有最大影響，但與保險人整體業績不一定一致
- 謹慎的保險人只在實際業績比精算假設更有利後才宣佈紅利
- 保險人一般保留釐定紅利金額的權利，由獲委任精算師建議、董事會批准

#### 三種紅利派發方式

| 方式 | 派發時間 | 特點 | 復歸權益？ |
|------|---------|------|-----------|
| **(a) 現金紅利** | 每年 | 可存放於保險人處滾存生息；**即時可用**；香港最常用 | **否**（不屬於復歸權益） |
| **(b) 復歸紅利** | 每年 | 加到保額上，**死亡/期滿時才支付**；保單利益隨宣佈金額不斷增加 | **是**（屬於復歸權益） |
| **(c) 期終額外紅利** | 期滿/特定期間後 | 與基金資產股值相若；**死亡/期滿時才支付**；可與(a)(b)同時使用 | **是**（屬於復歸權益） |

> **速記**：復歸 + 期終 = **等死亡/期滿才拿**；現金紅利 = **即時可用**，最靈活

#### 現金紅利的五種選擇

1. 立即以**現金**支付
2. 用以**繳付未來保費**（保費抵銷）
3. **存放在保險人處**賺取利息（紅利存款與現金價值是分開的）
4. 購買**清繳增額保險**（同樣會產生紅利）
5. 購買**為期一年的定期壽險**

#### 指引16對分紅保單的要求

| 要求 | 內容 |
|------|------|
| 公司政策 | 須有涵蓋股東與分紅基金間分配盈餘及宣佈紅利的公司政策，經董事會批准 |
| 精算師責任 | 確保向客戶展示的非保證回報有合理機會實現 |
| 紅利宣佈機制 | 須受保監局規管審查；保監局可要求委任獨立人士評估 |
| 董事會責任 | 最終負責解讀保單持有人的合理期望及就紅利宣佈作出決策 |
| 紅利實現率 | 須在公司網站披露每個近期曾發出新保單的產品系列的非保證紅利實現率 |
| 有效利益說明 | 須至少每年向保單持有人提供反映最新情況及未來展望的有效利益說明 |

⚠️ **易考點**：紅利實現率 = 實際派發的非保證紅利 / 銷售時利益說明所述數額的平均數。須提醒客戶過往紅利派發並非未來表現的指標。

### 5.2.9 財務需要分析（指引30）

**背景**：基於壽險保單的長期性質，保單所有人可能被鎖定資金，流動性嚴重受損。因此，作出建議前須確保對客戶的情況（包括需要、財務狀況、支付保費的能力及意願）作適當評估。

#### (a) 必須進行FNA的保單類別

| 必須FNA的保單 | 除外 |
|-------------|------|
| 類別A（人壽及年金） | (1) 定期人壽保險；(2) 純保障保單（住院/醫療/危疾/意外/殘疾/長護，不含高儲蓄成分的保費回贈或無現金價值的可續保保單）；(3) 團體保單 |
| 類別C（投連壽險） | 無例外 |

⚠️ **易考點**：定期壽險豁免FNA是因為它沒有儲蓄成分，不會鎖定資金。但類別C（投連壽險）必須做FNA，沒有例外。

**現有保單的新附加保障或額外保障**：須遵從FNA規定，猶如附加保障是一份新的人壽保險保單。

#### (b) 豁免產品（豁免FNA）的條件

須**同時符合**以下4項：

| 條件 | 內容 |
|------|------|
| (a) 產品類型 | (1) 非分紅儲蓄產品，保證款項於年期屆滿時≥已繳保費總額；**或** (2) 萬用壽險產品，退保價值於任何時間均≥已繳保費總額 |
| (b) 銷售渠道 | 由保險人或其代理機構透過**數碼分銷渠道**（網頁/手機App）直接銷售 |
| (c) 無建議 | 銷售前或期間**未向客戶提供建議** |
| (d) 披露 | 須作出5項披露：保單目標、類別及性質、目標保費/繳付期/保障期、負擔能力警告、流動性風險資料 |

❌ **易錯點**：豁免條件須**全部符合**才豁免。缺任何一項都須做FNA。即使產品本身符合(a)，但如果是通過中介人銷售（非數碼渠道）或提供了建議，仍須做FNA。

#### (c) 記錄及有效期

| 項目 | 規定 |
|------|------|
| 簽署 | 填妥的FNA表格須由客戶及中介人**簽署並註明日期** |
| 數碼簽署 | 可接受 |
| **有效期** | 簽署日起計**12個月**內持續有效 |
| 12個月內額外購買 | 毋須另一次FNA，**除非**客戶情況有重大變更或會造成錯配 |
| 客戶拒絕披露資料 | 中介人不得向客戶建議任何保險產品；保險人可能拒絕申請 |

⚠️ **易考點**：FNA有效期12個月。在同一保險人/中介人處12個月內購買額外保單，不用重做FNA（除非情況有重大變更或會造成錯配）。

#### (d) 建議前須考慮的事項

- 評估須基於客戶的**所有有效保單**（不局限於正在申請的保單）
- 須考慮客戶繳付保費的**能力及意願**（包括整個保費繳付期內）
- 如保費繳付期超逾客戶目標退休年齡，須評估退休後是否仍可負擔
- 如客戶擬使用**保費融資**，須以嚴格謹慎標準評估負擔能力
- 如客戶不願/不能選擇及管理投連壽險的投資選項，**不應**建議投連壽險

#### (e) 提供保險選擇

| 適用 | 規定 |
|------|------|
| 建議投連壽險時 | 須同時介紹能符合投資目標的**分紅保單**供選擇（指引15） |
| 持牌保險經紀建議產品時 | 須同時介紹最少一項由**不同保險人**提供的保險選擇；唯一例外是確實無其他選擇 |

#### (f) 處理錯配

**錯配的定義**：基於FNA過程中收集的資料顯示某保險產品**並不符合**客戶的需要和狀況，而持牌保險中介人仍向客戶建議該產品。

| 錯配類型 | 處理方式 |
|---------|---------|
| 一般錯配 | 須向客戶清楚解釋錯配情況及為何仍建議；須記錄解釋詳情 |
| 核保審視 | 保險人應在核保過程中審視及評估建議的合理性 |
| **保障水平偏差** | **不算錯配**（如偏差有恰當、良好及合理的依據，如承保限制或健康狀況）；須解釋並記錄 |
| **利益/保障期偏差** | 同上，不算錯配（如有合理依據） |
| **財富增值目標** | 如建議產品無法在客戶表明的時間內達到目標儲蓄金額，除非有合理依據，否則視為錯配 |
| **負擔能力錯配** | 一般**不應接受**（如目標繳付期10年但建議終身繳付產品）；少數情況可建議但須解釋並記錄 |

⚠️ **易考點**：
- 保障水平高於或低於FNA所識別的水平——**不算錯配**（如有合理依據）
- 保費繳付期**較客戶目標短**——**不算錯配**（只花更短時間繳清不算問題）
- 保費繳付期**較客戶目標長**（如目標10年但建議終身繳付）——**算錯配**，一般不應接受

## 5.3 核保

### 核保的本質

核保 = 為了提供保險或決定採用甚麼保險條款而進行的風險評估。

⚠️ **關鍵特點**：人壽保險涉及保險人**不能取消**的長期合約，通常壽險核保只能進行**一次**。第一次就正確評估風險非常重要。

### 5.3.1 核保因素

核保分為兩階段：(a) 確定風險程度；(b) 將建議風險分類。

#### (a) 確定風險程度——兩大主題

| 比較 | 實質危險（Physical hazard） | 道德危險（Moral hazard） |
|------|---------------------------|------------------------|
| **性質** | 客觀因素 | 主觀因素 |
| **內容** | 可能增加死亡發生率的客觀因素 | 準被保險人的基本誠信或正直意向 |
| **例子** | 明顯超重、煙吸得多、濫用物質（酒精/毒品）、高危職業或消閒活動、家族不良遺傳病歷或個人不良病歷 | 刻意隱瞞重要資料或提交虛假資料 |
| **判斷難度** | 較易確定（有客觀證據） | 較難確定（主觀判斷） |
| **自殺** | — | 通常不是潛在問題（很大程度上可透過保單條款解決，即自殺免責期） |

❌ **易錯點**：題目問「影響實質風險的事項」時，「申請人不誠實」屬於**道德危險**，不屬於實質危險。模擬試題中 (iii) 提供資料時申請人不誠實是道德危險，不是實質危險。

#### (b) 風險分類——四類

| 類型 | 特點 | 處理方式 |
|------|------|---------|
| **標準風險**（Standard risks） | 無異常特點，完全按適用於同年齡同性別者的保費接受 | 正常保費 |
| **次標準風險**（Sub-standard risks）/特殊風險（special class risks） | 預計產生比正常生命群組較高的死亡率 | 經特定考慮後仍可保；附加保費/負債/除外/限制計劃 |
| **拒保風險**（Declined risks） | 某特定保險人發覺不可接受的風險 | 拒保；保險人一般會盡量避免此激烈措施 |
| **優良風險**（Preferred risks） | 比平均風險優良（如經證實的非吸煙者） | 折扣或其他有利條款；並非所有保險人都採用此類別 |

#### 技術性核保 vs 財務性核保

| 類型 | 內容 |
|------|------|
| **技術性核保** | 對個別風險的潛在危險及預期危險進行評估（上述實質/道德危險） |
| **財務性核保** | 對投保金額作出評估，包括：(1)預期保單所有人履行繳費責任的能力；(2)所顯示的風險程度（再保險安排）；(3)就同一受保生命已累積購買的保單；(4)保額是否超出通常水平 |

### 5.3.2 醫療報告

#### 免體檢業務 vs 要體檢業務

| 類型 | 特點 |
|------|------|
| 免體檢業務 | 投保單上的資料及其他情況（年齡、表面健康狀況、保額等）容許核保人在毋須進一步調查下核保 |
| 要體檢業務 | 須合資格醫務人員提供進一步資料 |

#### 醫療報告來源

| 來源 | 定義 | 特點 |
|------|------|------|
| **主診醫生**（Attending Physician） | 通常或曾經向投保人提供醫療護理的醫生 | APS是最通常要求到的醫療報告 |
| **體檢醫生**（Examining Physician） | 在保險人要求下為投保人進行身體檢查的醫生 | 費用由**保險人**支付 |

#### 要求APS的常見理由

1. 須進一步調查投保單中披露的某些健康資料
2. 要求的保險金額很高
3. 投保人年齡較高（例如超過50歲）

⚠️ **易考點**：體檢醫生的費用由**保險人**支付，不是投保人支付。

#### 其他醫療調查

- **特殊醫療問卷**：針對投保單上填寫的特定狀況（如高血壓、癌症）發出特別設計的問卷
- **保密**：健康資料屬個人私隱，須按最高機密原則處理；投保人有權知道要進行甚麼檢查、目的及結果

### 5.3.3 次標準人壽及核保措施

針對次標準風險，可採取的核保措施：

| 措施 | 內容 | 適用情況 |
|------|------|---------|
| **(a) 拒絕提供保險（拒保）** | 保險人會盡量避免的激烈措施 | 風險完全不可接受 |
| **(b) 附加保費** | **最常用**的處理方法；將不正常風險轉化為可受保風險 | 一般次標準風險 |
| **(c)(i) 設置保單負債（debt/lien）** | 負債一般**逐年遞減**直至指定日期完全消失 | 超額死亡率明顯處於下降趨勢及屬臨時性質 |
| **(c)(ii) 指定除外責任** | 就某類非常危險的消閒活動引致的死亡提出除外條款 | 很少見（可能破壞投保原意） |
| **(c)(iii) 提供限制較多的計劃** | 如提供短期保險代替長期保險 | 醫療證據顯示非常長期的保險可能有問題 |
| **(c)(iv) 延遲決定** | 將決定延遲至投保人康復 | 投保人正嚴重受傷或處於暫時不利的健康狀況 |

#### 附加保費的「附加百分比表」

- 將次標準人壽按標準死亡率的預期百分比分為不同組別
- 對個別風險收取足以反映其超額死亡率的附加保費

#### 保單負債的計算（計分形式）

- 核保人給正常平均健康的生命打100分
- 有不良特點（如超重）的加分，有優良特點（如不抽煙）的減分
- 超過100分的部分稱為「過高死亡分數」
- 過高死亡分數變換成附加保費或減低保額的負債

**遞減負債示例**：
- 保額400,000元，年期20年儲蓄壽險，起保時負債190,000元
- 第一年死亡：保單收益 = 400,000 - 190,000 = 210,000元
- 首19年每年底負債降低10,000元，實質保障增加10,000元
- 最後一年保障 = 400,000元（負債歸零）
- 英式紅利按保額的全數（票面保障金額）分配

⚠️ **易考點**：遞減負債適用於超額死亡率**處於下降趨勢**的情況（如剛做過手術，風險隨時間降低）。如果超額死亡率是**固定或上升**的，應用附加保費。

## 5.4 簽發保單

⚠️ **核心原則**：簽發保單後，沒有保單所有人的同意**不能取消或修改**保單。對保險人而言，保單的簽發與交付幾乎是「欲罷不能」的境地，因此簽發前須仔細查核與確認。

### 5.4.1 交付保單

- 內部系統應建立客戶記錄並查證是否已收到首期保費
- 保單大多以標準形式印製，只有個別客戶的不同之處須更改標準格式
- 須查核保單文件確保無誤

## 5.6 理賠

### 三類索償

人壽保險中，如果保單持續有效，索償（定期保險除外）是不可避免的。大部分壽險合約帶有儲蓄成分，保單所有人通常都會期望進行索償。

索償分為三類：期滿索償、死亡索償、退保索償。

### 5.6.1 期滿索償

主要與儲蓄壽險有關——受保生命仍然生存並能親自收取保險金。

| 步驟 | 內容 |
|------|------|
| 接近到期日 | 到期日前約一個月發信：(i)提醒期滿日期；(ii)說明應付金額；(iii)列出支付條件；(iv)附上解除責任憑證(release) |
| 索償權利 | 只能是享有收取保險金權利的人：保單所有人本人、承讓人（如已轉讓）、受託人（如已成信託財產） |
| 出示保單 | 須出示保單；遺失保單不會十分重要（保單是合約的證據，不是合約本身），但保險人會要求宣誓及書面彌償 |
| 調整 | 須扣除：保單抵押貸款、未繳清保費、尚欠利息 |
| 年齡證明 | 如保單註明「年齡未獲承認」，須要求證實年齡 |

### 5.6.2 死亡索償

死亡索償在嚴肅甚至悲慘的情況下進行，保險人及中介人應寄予同情但不能過分干擾專業過程。

| 關注事項 | 詳情 |
|---------|------|
| **(a) 索賠權** | 有權獲取死亡利益的人：第三者保單的尚存保單所有人、受保保單所有人的遺產代理人、承讓人、受託人 |
| **(b) 死亡日期** | 須確定（影響遞減定期壽險的支付金額、紅利計算、保單是否已過期） |
| **(c) 死亡證明** | 一般須出示**正本**死亡證明書；外地死亡可能出現證明問題，容易發生欺詐 |
| **(d) 死因** | 見下表 |
| **(e) 推定死亡** | 不能簽發死亡證明書但可推定已死亡的情況，可能須由法庭裁決 |
| **(f) 年齡證明** | 出示出生證明書、身份證明文件或護照 |

#### 死因的重要性

| 死因 | 重要性 |
|------|--------|
| **自殺** | 發生於自殺免責期內（通常1年）→ 不賠；免責期後 → 如常賠付 |
| **意外** | 保單可能附加ADB附約，須確定是否意外死亡以決定是否額外賠付 |
| **可疑/令人驚訝** | 保單簽發後不久便死亡，或死因須持續比保單已生效時間更長才會引致死亡 → 須進一步調查；可能未過可異議期（2年） |
| **謀殺** | 大部分情況不影響索償有效性；**但若殺人犯是受益人，法律（「公共政策」）不容許殺人犯本人獲益** |

⚠️ **易考點（公共政策原則）**：殺人犯不能獲益——如果受益人殺死受保人，**公共政策**不容許殺人犯從其犯罪行為中獲益。這不是保單條款，而是法律原則。

❌ **易錯點**：
- 謀殺本身**不影響索償有效性**——保險人仍須賠付，只是不能賠給殺人犯受益人
- 死因在基本壽險中大部分不重要，但在以下情況重要：自殺條款、意外附約（ADB）、可疑索償調查、可異議期內

### 5.6.3 退保

由於索償人仍然在生，處理期滿索償的大部分問題都與退保索償有關。

| 關注事項 | 詳情 |
|---------|------|
| **(a) 所有權證明** | 有權收取現金價值的人：保單所有人、承讓人、受託人（或破產管理人） |
| **(b) 調整** | 須計算未繳清保費、保單費用、保單抵押貸款及利息 |
| **(c) 解約** | 須取得合適的解除責任憑證，妥善照顧承讓人或第三者的利益 |
| **(d) 其他調查** | 保險人/中介人應特別小心退保申請——可能有意圖的欺詐行為（如**洗黑錢**） |

⚠️ **易考點**：
- 退保須防範**洗黑錢**等欺詐行為
- 客戶要求退保**未必符合其最佳利益**——保險人/中介人可建議其他更合適的選擇：
  - **保單抵押貸款**（利用現金價值貸款，保單繼續有效）
  - **不能作廢條款**（減額清繳保險 / 展期保險 / 自動保費貸款）
- 慎重而有禮的查問可能有助看破欺詐意圖

❌ **易錯點**：退保是保單所有人的權利，保險人不能拒絕。但保險人/中介人有責任提醒客戶其他選擇，並對可疑退保進行調查。

## 第5章考試答題要點

1. **冷靜期**：21個曆日（不是14天），9個曆日交付限期，第21天非工作天順延。不是保險人可以取消保單的時間，是保單持有人可以取消的時間。
2. **TIA vs 附條件收據**：TIA是獨立合約，不以可保性為條件；附條件收據須事後確定可保性。
3. **轉保**：核心是「用舊保單的錢買新保單」。轉換權行使不算轉保。內部轉保雙向12個月核對。
4. **利益說明**：投資壽險用0/3/6/9%（或0/3/6%）；分紅保單用悲觀（25th）/樂觀（75th）百分位數，必須同頁顯示；萬用壽險用保證基礎+現時假設基礎。
5. **FNA**：類別A（除定期/純保障/團體）及類別C必須做。豁免須同時符合4條件。12個月有效。保障水平偏差不算錯配。負擔能力錯配一般不應接受。
6. **內地人士**：7個工作日送回投保書（不是曆日）、12號字體、每頁簽字、簡體中文版、逐項細閱。
7. **核保**：實質危險（客觀）vs 道德危險（主觀）。四類風險：標準/次標準/拒保/優良。次標準措施：附加保費（最常用）、遞減負債（適用下降趨勢風險）、除外責任、限制計劃、延遲決定。
8. **死亡索償**：殺人犯不能獲益（公共政策）；自殺免責期內不賠；死因對ADB附約重要。
9. **退保**：防洗黑錢；未必符合客戶最佳利益，可建議保單抵押貸款/不能作廢條款。
10. **團體保單**：不適用冷靜期（指引29）、不適用轉保指引（指引27）、FNA不適用（指引30）。

## 一、不可異議條款（10題，高頻陷阱）

### ⚠️ 易考點
- 可異議期 = **2年**（不是1年、不是3年）
- 必須**受保生命在世時**度過2年（受益人不能故意延遲索償至可異議期結束）
- **欺詐例外**：欺詐行為可推翻不可異議條款

### ❌ 易錯點
1. "保單生效兩年後，即使有欺詐也不可異議" → **錯！** 欺詐可推翻
2. "保單首兩年是不可異議期" → **錯！** 首兩年是**可異議期**，之後才不可異議
3. "違反最高誠信原則，兩年後仍可拒賠" → **看情況**：
   - 故意隱瞞（欺詐）→ 可拒賠
   - 投保人真誠認為資料無關（非欺詐）→ **不可拒賠**（Q472答案=D）
4. "兩年後保單仍可異議的原因" → 只有**欺詐**或**受保人在2年內已死亡**
5. 不可異議條款**不防止**保險人提出合約非法（如缺乏可保權益）

### 考題模式
- "以下哪項陳述是不正確的？" → 答案通常是"欺詐也不可異議"這個錯誤說法
- "兩年後仍可異議是因為？" → 答案：涉及欺詐

---

## 二、ADB / 雙倍賠償（40題，最多！）

### ⚠️ 易考點
- ADB = **額外**再賠一筆等於基本保額的錢 → 總賠付 = 2×保額
- "Double Indemnity"這個英文名**技術上不準確**（Q265）
- 死亡須於意外後**1年內**發生
- **近因**對ADB非常重要（基本壽險中死亡原因一般不重要）
- ADB也受**1年自殺免責期**約束（與基本保單一樣）
- 同一意外：先喪失肢體後死亡 → **只支付死亡保險金**，不重複賠付

### ❌ 易錯點
1. "ADB本身就是雙倍賠償" → **錯！** 是主保單賠1份 + ADB額外賠1份 = 2份
2. "喪失肢體和死亡可同時賠付" → **錯！** 同一事故只赔一項
3. "意外死亡不適用自殺免責期" → **錯！** ADB同樣受1年免責期約束
4. "喪失一肢 = 全額賠付" → **錯！** 喪失**兩肢**或**雙目失明**才全額；喪失單肢/單目 = 規定比率
5. ADB除外責任包括：**蓄意自殘（含自殺）、戰爭、違法活動、航空（作為購票乘客除外）**

### 喪失肢體賠付比例
| 情況 | 賠付 |
|------|------|
| 喪失兩肢 / 雙目失明 | = 全額（等同意外死亡保險金） |
| 喪失單肢 / 單目失明 | 規定比率（低於全額） |
| 同一事故先喪失肢體後死亡 | 只赔死亡保險金 |

## 三、可保權益（31題，高頻）

### ⚠️ 易考點
- 只在**合約開始時**需要有可保權益
- 對自己、對配偶 = **無限金額**，毋須證明
- 第64A條：父母/監護人對**未成年人**（<18歲）有可保權益（配偶以外唯一血緣）
- 第64B條：缺乏可保權益 → 合約**無效**
- **轉讓**：正當安排可轉讓給缺乏可保權益的第三者

### ❌ 易錯點
1. "離婚後保單失效" → **錯！** 可保權益只在合約開始時需要，離婚後仍有效
2. "債務償還後保單失效" → **錯！** 同上
3. "兄弟姐妹之間有可保權益" → **錯！** 香港只有配偶 + 父母對未成年子女
4. "為繞過可保權益規定而轉讓是有效的" → **錯！** 屬無效
5. "購買別人的現有保單需要可保權益" → **錯！** 透過轉讓程序購買，無須可保權益
6. "可保權益金額不限" → 只對自己和配偶成立；對其他人須可用金錢估值

## 四、寬限期（13題）

### ⚠️ 易考點
- **30或31日**（不是15日、不是60日）
- **不適用於首期保費**
- 寬限期內繳付 = 視作準時
- **並非免費保險期**：寬限期內死亡，從保險金扣除到期保費

### ❌ 易錯點
1. "寬限期是免費保險期" → **錯！** 死亡會從保險金扣保費
2. "寬限期適用於首期保費" → **錯！** 只適用續期保費
3. 英式 vs 美式失效時間：
   - **英式**：保費到期日即失效（過期當天）
   - **美式**：寬限期屆滿才失效（多送一個月）

## 五、自殺除外責任（16題）

### ⚠️ 易考點
- 免責期 = **通常1年**
- 免責期後自殺 → **如常賠付**
- 免責期內自殺 → 不賠，退還已付保費（減去未繳貸款及利息）
- **舉證責任在保險人**（不是受益人）

### ❌ 易錯點
1. "所有人壽保險都不赔自殺" → **錯！** 過免責期後自殺正常賠付
2. "舉證責任在受益人" → **錯！** 在保險人
3. "免責期內自殺不退保費" → **錯！** 退還已付保費（扣欠款）
4. 復效後有**另一段新的自殺免責期**

---

## 六、轉讓（27題，高頻！）

### ⚠️ 易考點
- 轉讓自向保險人發出**書面通知**起生效
- **絕對轉讓**：不可撤銷，轉讓一切擁有權
- **抵押轉讓**：暫時性，限於貸款額+利息，還款後取回
- 保險人可向承讓人執行**抵銷權**（扣過期保費、未償貸款）→ 承讓人收**淨保單收益**
- 轉讓**不得侵犯不可撤換受益人**的既得權利

### ❌ 易錯點
1. "轉讓自簽署轉讓書起生效" → **錯！** 自向保險人發出書面通知起生效
2. "抵押轉讓後轉讓人仍可退保" → **錯！** 已通知的抵押轉讓生效後不能退保/保單抵押貸款
3. "承讓人可獲得比轉讓人更多的利益" → **錯！** 承讓人繼承一切權利但不可能比轉讓人收得多
4. "轉讓可侵犯不可撤換受益人權利" → **錯！** 不得侵犯
5. 容許轉讓的法律 = 《法律修訂及改革（綜合）條例》第9條（不是《保險業條例》）

---

## 七、不能作廢條款（23題）

### ⚠️ 易考點
- 適用於大多數傳統壽險（**定期壽險除外**）
- 保費中斷時三個選擇 + 一個自動條款

### ❌ 易錯點
1. "定期壽險也有不能作廢條款" → **錯！** 定期壽險除外
2. 三個選擇的區別：
   - 退保 → 中止保單，取現金
   - 減額清繳 → **同計劃、保額降低**、整付
   - 展期 → **同保額**、變定期壽險、期限取決於現金價值
3. "自動保費貸款（APL）是三個選擇之一" → **不是！** APL是獨立的常規規定，用現金價值自動墊繳
4. 沒有APL條款的保單 → 默認採用**展期保險**
5. 有APL條款的保單 → 默認採用**自動保費貸款**

## 八、復效（16題）

### ⚠️ 易考點
- 只適用於**已失效**保單，**不適用於已退保**
- 期限：一般**5年**內
- 復效後有**另一段可異議期** + **另一段自殺免責期**

### ❌ 易錯點
1. "已退保的保單可以復效" → **錯！** 只有已失效（lapsed）可以
2. "復效後原來的可異議期繼續" → **錯！** 從復效日計起**另一段新的**可異議期
3. 復效條件**全部**都要滿足：可保性證明 + 繳清貸款連息 + 補付保費連息 + 復效費 + 新可異議期 + 新自殺免責期

---

## 九、最高誠信 / 披露責任（12題）

### ⚠️ 易考點
- 重要事實 = 影響審慎保險人判斷的任何情況
- **與索償疾病無關**的重要事實也須披露
- **免體檢**投保：保險人不能指稱投保人不披露申請表沒問的問題
- **要體檢**投保：醫療人員的疏忽遺漏不歸咎投保人

### ❌ 易錯點
1. "只有與索償疾病相關的事實才須披露" → **錯！** 無關的也要披露
2. "免體檢投保就不用披露" → **錯！** 仍須披露申請表問的問題，只是不用披露沒問的
3. "要體檢就不用披露病歷" → **錯！** 體檢不等於充分披露
4. "投保人不知道的疾病也算不披露" → **錯！** 有理由不可能知道的不須披露
