# 第1章 人壽保險簡介 (10%)

## 1.1 壽險定義與需求

- **定義**：保險公司承諾當被保險人死亡時支付保險金
- **最初目的**：被保險人早逝時向家人提供經濟幫助
- **個人需求**：受養人生活開支、善終開支、教育經費、退休收入、償還按揭、應急資金、殘疾收入
- **業務需求**：關鍵人物、企業擁有人、合夥人、僱員福利

## 1.2 壽險原則

| 原則 | 人壽險適用情況 |
|------|---------------|
| 可保權益 | ✅ 只在合約開始時需要 |
| 最高誠信 | ✅ 主動披露所有重要事實 |
| 近因 | ⚠️ 應用有限（死亡原因一般不重要，自殺條款/意外附約時才重要） |
| 彌償 | ❌ 不適用（人壽險是**利益保單**，非彌償保單） |
| 分擔 | ❌ 不適用（可有多份保單，每份全額賠償） |
| 代位權 | ❌ 不適用（賠付後無權向第三者追討） |

### 可保權益

| 條文 | 內容 |
|------|------|
| **第64A條** | 父母/監護人對未成年人（<18歲）有可保權益（配偶以外**唯一**血緣關係） |
| **第64B條** | 缺乏可保權益的合約屬**無效** |
| **第64C條** | 具權益的人名字須寫入合約 |

- **對自己及配偶**：無限金額，毋須證明
- **對其他人**：須具可用金錢估值的權益（債務人、合夥人、關鍵人物）
- **何時需要**：只在合約開始時 → 離婚後、債務償還後保單仍有效
- **轉讓**：正當安排可轉讓給缺乏可保權益的第三者；為繞過規定而轉讓屬無效

### 披露責任（最高誠信）

- **重要事實**：影響審慎保險人判斷的任何情況
- **毋須披露**：有理由不可能知道的、申請表沒問的（免體檢）、醫療人員疏忽遺漏
- **關鍵原則**：與索償疾病**無關**的重要事實也須披露

## 1.3 保費計算

**三大因素**：
- 死亡率 + 利息 = **淨保費**（純保險費）
- 淨保費 + 附加保費（開支） = **毛保費**

| | 自然保費制度 | 均衡保費制度 |
|---|---|---|
| 保費 | 隨年齡增 | 固定不變 |
| 現金價值 | 無 | 有 |
| 逆選擇 | 易受影響 | 不受影響 |
| 現狀 | 已淘汰 | **主流** |

**均衡保費產生的特性**：現金價值/退保價值、保單抵押貸款、不能作廢條款、清繳保險

# 第2章 人壽保險及年金的種類 (20%)

## 2.1 傳統壽險三類

| | 定期壽險 | 儲蓄壽險 | 終身壽險 |
|---|---|---|---|
| 保障期 | 指定期限 | 指定期限 | 終身（至99/100歲） |
| 現金價值 | **無** | 有 | 有 |
| 保費 | **最便宜** | 較貴 | 中等 |
| 賠付 | 期限內死亡 | 死亡或期滿 | 任何時候死亡 |

### 定期壽險變體

| 類型 | 保額 | 特點 |
|------|------|------|
| 定額定期 | 不變 | 最流行 |
| 遞減定期 | 按年遞減 | **最便宜**；信用壽險/抵押贖回/家庭收入 |
| 遞增定期 | 按年遞增 | 抗通脹 |
| 可續保 | 不變 | **毋須健康證明**，按到達年齡計費 |
| 可轉換 | 不變 | **毋須健康證明**轉永久計劃，通常55/65歲後不能轉 |

- **抵押贖回保險 vs 信用壽險**：前者保按揭人權益（利益保險），後者保貸款人權益（彌償保險）

### 儲蓄壽險
= 定期壽險 + 純生存保險。必定給付（死亡或期滿），保費不便宜，香港不流行。

## 2.2 非傳統壽險

### 萬用壽險四大特徵
1. **靈活保費**（現金價值不足60天費用時失效）
2. 可調整死亡保險金
3. 分別列示定價（保障成本/利息/開支）
4. 累積現金價值

### 單位相連長期保險
保單價值直接與投資表現相連，**價值可升可跌**。

## 2.3 年金

| | 即期年金 | 延期年金 |
|---|---|---|
| 購買 | 整付 | 整付/分期 |
| 開始支付 | 即時 | 累積期→年金期 |

**年金變化**：確定年金（不論生死）、終身年金、臨時終身年金、確定期間終身年金（保證期+終身）

**核保特點**：年金以**生存率**為基礎（年齡越高每期年金越高，男性高於女性）—— 與壽險**相反**

### 香港年金計劃（考試常考）

| 項目 | 數字 |
|------|------|
| 推出日期 | 2018年7月5日 |
| 年齡 | ≥65歲香港永久性居民 |
| 保費範圍 | $50,000 - $3,000,000 |
| 保證期 | 累計年金達已繳保費**105%**為止 |
| 特別款項提取上限 | $300,000（只可一次） |

### 延期年金稅務扣除（QDAP）

| 項目 | 數字 |
|------|------|
| 生效日期 | 2019年4月1日 |
| 每人年扣稅上限 | $60,000 |
| 夫婦合計 | $120,000 |
| 最高節稅 | $10,200/年（17%） |
| 最低保費總額 | $180,000 |
| 最短繳付期 | 5年 |
| 最短年金期 | 10年 |
| 年金最早開始年齡 | 50歲 |
| 無固定年期IRR假設 | 30年 |

## 2.4 團體保險

- 一份保單保障多人（總團體保險合約）
- **集體核保**，不要求詳細個人資料
- **在職工作條款**：生效時須在職
- 可轉為個人保單（毋須可保性證明，限指定時期）
- **不適用冷靜期（指引29）、不適用轉保指引（指引27）**

# 第4章 闡釋人壽保險單 (24%)

## 條款速覽

### 4.1 完整合約條款
保單+附約+投保單複印本 = 完整合約。只有保險公司**指定高級職員**書面更改才有效。

### 4.2 不可異議條款
- **可異議期 = 2年**（受保生命在世時）
- 防止受益人故意延遲索償至可異議期結束
- **欺詐例外**：欺詐不受保護
- 不防止保險人提出合約非法（如缺乏可保權益）

### 4.3 寬限期
- **30或31日**，不適用於首期保費
- 寬限期內繳付視作準時
- **並非免費保險期**（死亡會從保險金扣除到期保費）
- 英式：保費到期日失效；美式：寬限期屆滿失效

### 4.4 受益人指定
- **可撤換**：可隨時變更
- **不可撤換**：須受益人書面同意才能變更
- 為配偶/子女買壽險 = **法定信託**（《已婚者地位條例》），實益權益不可收回

### 4.5 不能作廢條款（定期壽險除外）

保費中斷時三個選擇：

| 選擇 | 內容 |
|------|------|
| 退保現金價值 | 中止保單 |
| 減額清繳保險 | 同計劃、保額較低、整付 |
| 展期保險 | 同保額定期壽險、期限取決於現金價值 |
| 自動保費貸款(APL) | 現金價值自動墊繳（有此條款時默認） |

### 4.6 保單抵押貸款
- 貸款上限 = 現金價值 - 1年利息
- 用途無限制
- 未償還貸款連利息從死亡保險金/退保價值扣除

### 4.7 復效
- 須於**5年**內提出
- 只適用於**已失效**保單，**不適用於已退保**
- 條件：可保性證明 + 繳清貸款連息 + 補付保費連息 + 復效費 + **另一段可異議期** + **另一段自殺免責期**

### 4.8 誤報年齡/性別
- 索償後發現：調整利益金額（英式只**向下**調整，多繳保費退還不含利息）
- 索償前發現：可調整保額或退費

### 4.9 轉讓

| | 絕對轉讓 | 抵押轉讓 |
|---|---|---|
| 性質 | 不可撤銷 | 暫時性 |
| 權益 | 一切擁有權 | 限於貸款額+利息 |
| 還款後 | — | 取回權益 |
| 保單抵押貸款/退保 | — | 不能進行 |

轉讓自向保險人發出**書面通知**起生效。

### 4.10 紅利選擇
三種紅利：現金紅利（每年）、復歸紅利（每年）、期終額外紅利（期滿）
現金紅利選項：現金/繳保費/存放生息/買清繳增額保險/買一年定期壽險

### 4.11 賠付選擇
整筆支付 / 利息選擇 / 定期支付 / 定額支付 / 終身收益選擇

### 4.12 自殺除外責任
- **免責期 = 通常1年**
- 免責期後自殺：如常賠付
- 免責期內自殺：不賠，退還已付保費（減去未繳貸款及利息）
- **舉證責任在保險人**

# 第5章 人壽保險程序 (22%)

## 5.1 公司運作

### 兩類保險公司
| 類型 | 特點 |
|------|------|
| **相互保險公司** | 無股東，由分紅保單持有人擁有，利潤不與股東分享 |
| **營業公司（股份公司）** | 由股東擁有的有限責任公司，更普遍的結構 |

### 七個部門職責一覽

| 部門 | 核心職責 | 舉例 |
|------|----------|------|
| **(a) 會計部** | 收入與支出記錄、財務報告；法例要求每年提交經審核財務報告 | 編製年報、管理帳目 |
| **(b) 精算部** | 產品定價、估值（影響償付準備金及紅利）、理賠及再保險計算、管理報告 | 計算保費、預測死亡率、決定紅利 |
| **(c) 代理人培訓與管理** | 培訓計劃、考試（IIQAS等）、資源與設施 | 組織代理人考試、培訓課程 |
| **(d) 理賠部** | 日常行政、各類索償（死亡/期滿/退保）、調查工作 | 審核死亡索償、調查可疑索償 |
| **(e) 顧客服務（POS）** | 通訊、文件整理、保單續保、**更改保單**、**保險權益管理**、收取保費 | 改地址、改受益人、保額變更 |
| **(f) 市場行銷** | 產品研究開發、促銷宣傳、廣告、公共關係、市場調查 | 推出新產品、品牌推廣 |
| **(g) 核保** | 風險評估、體檢要求、再保險安排 | 決定接受/拒保/加費 |

> **考點**：POS（售後服務）= 顧客服務部，負責**售後**事務（更改保單、權益管理、續保），**不是**索償（屬理賠部），也**不是**廣告（屬市場行銷）。

## 5.3 核保

### 風險分類

| 類型 | 特點 | 處理 |
|------|------|------|
| 標準風險 | 無異常 | 正常保費 |
| 次標準風險 | 死亡率較高 | 附加保費/負債/除外/限制計劃 |
| 拒保風險 | 不可接受 | 拒保 |
| 優良風險 | 優於平均（非吸煙者） | 折扣 |

### 核保因素
- **實質危險**：客觀（健康、職業、生活習慣）
- **道德危險**：主觀（誠信、正直意向）
- **財務性核保**：繳費能力、已累積保單、保額是否超常

### 次標準核保措施
附加保費（最常用）、設置遞減負債、指定除外責任、提供限制較多計劃、延遲決定

## 5.6 理賠

### 死亡索償
- **殺人犯不能獲益**：若受益人殺死受保人，**公共政策**不容許殺人犯獲益
- **自殺**：免責期內不賠
- **近因**：確定死亡原因對雙倍賠償/意外附約重要

### 退保
- 須防範欺詐（洗黑錢）
- 客戶要求退保未必符合最佳利益，可建議保單抵押貸款/不能作廢條款等

## 考試答題技巧

1. **注意否定字眼**：「不正確」「不屬於」「不可能」
2. **乙類多選題**：逐條判斷 i/ii/iii/iv 正誤，再組合
3. **壽險 vs 一般保險**：彌償/分擔/代位權**不適用**於壽險
4. **可保權益**：只在合約開始時需要（離婚/還債後仍有效）
5. **披露責任**：與索償疾病無關的重要事實也須披露
6. **近因**：壽險中很少重要（除非自殺條款/意外附約）
7. **殺人犯不能獲益**：公共政策原則
8. **團體保險**：不適用冷靜期、不適用轉保指引
9. **復效 vs 退保**：復效只適用於已失效，不適用於已退保
10. **不可異議條款**：2年 + 受保生命在世時 + 欺詐例外
